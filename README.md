## 一、考试定位

杭州区县事业编考试（临安、萧山、余杭、富阳、淳安、临平、桐庐、建德等）**不分类**，所有岗位统一试卷，区别于浙江省统考 ABCDE 五类。科目为《综合基础知识》或《综合基础知识与写作》。

- 单科考试，时长 2–2.5 小时
- 偏重记忆与基础知识
- 题型固定：单选 30×1 + 多选 10×2 + 判断 10×1 + 主观题 30-40 分

详见 [reference/00-考试说明/考试大纲与备考策略.md](./reference/00-考试说明/考试大纲与备考策略.md)

## 四、备考路径

### 阶段 1：基础打底（4–6 周）

1. 通读 1 个模块的 `知识点.md`，配合 `速记口诀.md` 背诵
2. 用 `易混对比.md` 自测易错点
3. 完成 `高频真题.md` 该模块练习

### 阶段 2：强化突破（3–4 周）

1. 攻克各模块 `易混对比.md`，消灭易错点
2. 刷 `高频真题.md` 全套，复盘错题
3. 写作模块每日练 1 篇，套用 [答题模板.md](./答题模板与速记卡/答题模板.md)
4. 时效性模块刷新 `2026更新.md`

### 阶段 3：冲刺模考（2 周）

1. 整套真题限时模考（用 [题库/](./题库/) 三套卷）
2. 反复过 [速记卡.md](./答题模板与速记卡/速记卡.md)
3. 重点回看 ⭐ S 级考点
