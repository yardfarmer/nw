# 杭州事业编学习应用

2026年浙江杭州事业编考试备考学习应用。

## 项目目标

构建一个帮助用户高效备考 2026 年浙江省杭州市事业单位公开招聘考试的学习工具，覆盖时政、省情、政治理论、通识知识等核心科目。

## 考试说明

本应用面向杭州区县单独招聘的事业编考试（如临安、萧山、余杭等），此类考试**不分类**（区别于浙江省统考 ABCDE 五类），所有岗位统一试卷，科目为《综合基础知识》或《综合基础知识与写作》。

### 考试科目特点

- 只考一科，时间约 2-2.5 小时
- 偏重记忆与基础知识，非分类应用分析
- 部分试卷包含主观题（材料写作 / 策论文，约 30-40 分）

### 考试范围

- **时政核心**：二十届三中全会、重要会议决议、国内国际重大事件
- **浙江专项**：浙江省情、浙江省政策与地方时政（杭州为重点）
- **政治理论**：马原、毛中特、习思想、党史党务
- **通识知识**：管理、文史、公文、经济、法律、科技、地理
- **政府工作报告与时事热点**：2025-2026 年政府工作报告要点、年度热点
- **写作**：材料写作 / 策论文

## 参考资料目录

知识库存放在 `reference/` 目录，按考试模块分文件，按考试权重排序：

| 文件 | 模块 | 权重 |
|------|------|------|
| `01-时政热点.md` | 二十届三中全会、二十大、政府工作报告、科技成就、重要数字 | ~25% |
| `02-政治理论.md` | 马原、毛中特、习思想、党史党务 | 含于政治 |
| `03-法律常识.md` | 宪法、民法、刑法、行政法、劳动法、公务员法 | ~20% |
| `04-经济常识.md` | 宏观/微观经济学、中国经济热点、政治经济学 | ~12% |
| `05-公文知识.md` | 15 种公文、文种辨析、格式规范、行文规则 | ~10% |
| `06-管理常识.md` | 管理理论、激励理论、公共管理、组织结构 | ~6% |
| `07-文史常识.md` | 中国历史、文学、文化、世界历史 | ~9% |
| `08-科技地理.md` | 航天成就、基础科学、中国地理、世界地理 | ~9% |
| `09-浙江省情杭州市情.md` | 八八战略、共同富裕示范区、数字经济、杭州特色 | ~4% |
| `10-写作热点.md` | 策论文框架、2026 热点话题、金句储备 | 主观题 |

`reference/` 为知识库源文件，可由 AI 更新补充。标注 ⭐ = 必背，🔥 = 超级重点。

## 约定

- 中文为本项目的主要语言，UI 文案、注释、commit 信息均使用中文
- 题目与知识点的数据结构以 JSON 或 TypeScript 类型定义，放在 `src/data/` 或 `src/types/`
- 如使用前端框架，默认 React + TypeScript + Vite
- 样式优先使用 Tailwind CSS
- 测试覆盖率目标 80%
- 代码文件不超过 800 行，函数不超过 50 行

## 数据标注约定

参考资料中使用的标注符号：

- ⭐ = 必背 / 高频考点
- 🔥 = 超级重点
- ※ = 易出题点

生成题库或卡片时应优先覆盖带标注的内容。

## 开发原则

1. 以学习效果为核心，功能设计服务于记忆和理解
2. 支持碎片化学习场景（移动端适配）
3. 题目练习 + 知识点回顾双路径
4. 进度可追踪、薄弱环节可识别
5. 离线可用，核心功能不依赖网络
