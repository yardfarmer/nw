## 第一篇：马克思主义基本原理

### 一、唯物论

#### 1.1 物质与意识

| 概念 | 要点 |
|------|------|
| 物质 | 物质的唯一特性是**客观实在性** |
| 意识 | 物质世界长期发展的产物；人脑的机能；客观存在的主观映像 |
| 关系 | 物质决定意识，意识对物质有能动的反作用 |
| 方法论 | 一切从实际出发，实事求是 |

#### 1.2 运动与静止

| 概念 | 内涵 |
|------|------|
| 运动 | 物质的根本属性和存在方式；**绝对的** |
| 静止 | 运动的特殊状态；**相对的** |
| 关系 | 动中有静，静中有动 |

#### 1.3 时间与空间

- 时间：物质运动的**持续性**和**顺序性**
- 空间：物质运动的**广延性**和**伸张性**
- 特性：时间一维（不可逆），空间三维

### 二、唯物辩证法（S 级必背⭐）

#### 2.1 两大总特征

| 特征 | 内涵 |
|------|------|
| **联系** | 事物内部各要素之间和事物之间相互影响、相互制约、相互作用的关系 |
| **发展** | 前进的、上升的运动；实质是**新事物的产生和旧事物的灭亡** |

##### 联系的特性

1. 客观性：联系是事物本身固有的
2. 普遍性：任何事物都与周围事物有联系
3. 多样性：直接/间接、内部/外部、本质/非本质等
4. 条件性：任何联系都依赖于一定条件

##### 发展的观点

- **新事物**：符合发展规律、具有强大生命力和远大前途
- **旧事物**：丧失了存在必然性、日趋灭亡
- 发展的实质：新事物的产生和旧事物的灭亡

#### 2.2 三大规律（必背⭐）

| 规律 | 揭示 | 核心内容 |
|------|------|----------|
| **对立统一规律** | 事物发展的**动力和源泉** | 矛盾是事物发展的根本动力；矛盾的基本属性是**同一性**和**斗争性** |
| **量变质变规律** | 事物发展的**状态和形式** | 量变是质变的必要准备，质变是量变的必然结果；质变为新的量变开辟道路 |
| **否定之否定规律** | 事物发展的**方向和道路** | 螺旋式上升、波浪式前进；方向是前进的，道路是曲折的 |

##### 对立统一规律（核心⭐）

- **矛盾**：事物内部包含的既对立又统一的关系
- 矛盾的**基本属性**：
  - **同一性**：矛盾双方相互依存、相互转化
  - **斗争性**：矛盾双方相互排斥、相互对立
- **内因**是变化的**根据**（根本原因）
- **外因**是变化的**条件**
- 内因决定外因，外因通过内因起作用

##### 矛盾的普遍性与特殊性（高频🔥）

| 类别 | 内涵 | 方法论 |
|------|------|--------|
| 普遍性 | 矛盾无处不在、无时不有 | 承认矛盾、分析矛盾 |
| 特殊性 | 不同事物的矛盾各有其特点 | **具体问题具体分析** |

特殊性表现：

1. 不同事物的矛盾各有特点
2. 同一事物的矛盾在不同发展阶段各有不同
3. 构成事物的诸多矛盾及每一矛盾的不同方面各有不同性质、地位和作用

##### 主要矛盾与次要矛盾

- **主要矛盾**：在复杂事物中处于支配地位、对事物发展起决定作用的矛盾
- **次要矛盾**：处于从属地位的矛盾
- **矛盾的主要方面**：在事物内部居于支配地位、起主导作用的方面
- 方法论：**抓重点、抓关键**，统筹兼顾

##### 量变质变规律

- **量变**：事物数量的增减和场所的变更（渐进的、不显著的变化）
- **质变**：事物根本性质的变化（连续性的中断）
- 关系：
  - 量变是质变的**必要准备**
  - 质变是量变的**必然结果**
  - 质变巩固量变成果，为新的量变开辟道路
- 方法论：重视量的积累；不失时机促成质变

##### 否定之否定规律

- **辩证否定**：事物自身的否定，是联系环节和发展环节
- 否定的特点：**自我否定**、**发展环节**、**联系环节**
- 实质：**扬弃**（既克服又保留）
- 过程：肯定 → 否定 → 否定之否定（循环往复，螺旋上升）

#### 2.3 唯物辩证法五对范畴

| 范畴 | 关系 |
|------|------|
| 原因与结果 | 前因后果，因果具有客观性、普遍性 |
| 必然性与偶然性 | 必然性存在于偶然性之中，偶然性为必然性开辟道路 |
| 可能性与现实性 | 可能性包含在现实中，现实性由可能性发展而来 |
| 内容与形式 | 内容决定形式，形式反作用于内容 |
| 本质与现象 | 本质通过现象表现，现象反映本质 |

### 三、认识论

#### 3.1 实践与认识

| 维度 | 实践 | 认识 |
|------|------|------|
| 地位 | **基础** | 反映实践 |
| 关系 | 实践决定认识 | 认识反作用于实践 |

##### 实践是认识的基础（4 点⭐）

1. 实践是认识的**来源**
2. 实践是认识发展的**动力**
3. 实践是检验认识真理性的**唯一标准**
4. 实践是认识的**目的**

#### 3.2 认识的规律

- **感性认识**：感觉、知觉、表象（直接、具体）
- **理性认识**：概念、判断、推理（间接、抽象）
- 关系：感性是理性的基础，理性是感性的升华

##### 认识的反复性与无限性

- **反复性**：由于客观条件限制，认识需要多次反复
- **无限性**：客观世界不断发展，认识永无止境
- **上升性**：认识不断深化、扩展、向前推移

#### 3.3 真理

| 特性 | 内涵 |
|------|------|
| **客观性** | 真理的内容是客观的；检验标准是客观的（实践） |
| **绝对性** | 任何真理都包含不依赖于人的客观内容 |
| **相对性** | 真理是有限的、近似的、有条件的 |
| **具体性** | 任何真理都是具体的、历史的 |

### 四、唯物史观

#### 4.1 社会存在与社会意识

| 概念 | 内涵 |
|------|------|
| **社会存在** | 社会物质生活条件的总和（地理环境、人口因素、物质资料生产方式） |
| **社会意识** | 社会精神生活的总和（政治法律思想、道德、艺术、宗教、哲学等） |
| 关系 | 社会存在决定社会意识，社会意识反作用于社会存在 |

##### 社会意识的相对独立性

1. 与社会存在变化发展不完全同步
2. 在发展上具有历史继承性
3. 各种社会意识之间相互影响
4. 对社会存在具有能动的反作用（最突出表现）

#### 4.2 社会基本矛盾

| 矛盾 | 关系 |
|------|------|
| **生产力与生产关系** | 生产力决定生产关系，生产关系反作用于生产力 |
| **经济基础与上层建筑** | 经济基础决定上层建筑，上层建筑反作用于经济基础 |

##### 生产力三要素

- 劳动者（最活跃的因素）
- 劳动资料（生产工具是标志）
- 劳动对象

##### 生产关系三方面

- 生产资料所有制（基础）
- 人们在生产中的地位和关系
- 产品分配形式

#### 4.3 人民群众

- 人民群众是**历史的创造者**
- 表现：
  1. 人民群众是**物质财富**的创造者
  2. 人民群众是**精神财富**的创造者
  3. 人民群众是**社会变革**的决定力量
- 方法论：群众路线（一切为了群众，一切依靠群众，从群众中来，到群众中去）

## 第二篇：毛泽东思想

### 一、毛泽东思想的形成与发展

| 阶段 | 时间 | 标志 |
|------|------|------|
| 萌芽 | 大革命时期 | 中国社会各阶级分析 |
| 形成 | 土地革命前中期 | 农村包围城市、武装夺取政权 |
| 成熟 | 遵义会议后到抗战 | 新民主主义革命理论 |
| 继续发展 | 解放后 | 社会主义改造理论 |

### 二、毛泽东思想活的灵魂（必背⭐）

| 要素 | 地位 | 内涵 |
|------|------|------|
| **实事求是** | 根本出发点和方法论 | 一切从实际出发，理论联系实际 |
| **群众路线** | 根本工作路线 | 一切为了群众，一切依靠群众，从群众中来，到群众中去 |
| **独立自主** | 根本立场 | 自力更生，依靠自己力量 |

### 三、新民主主义革命理论

#### 3.1 总路线

无产阶级领导的，**人民大众**的，反对**帝国主义、封建主义和官僚资本主义**的革命

#### 3.2 三大法宝（必背⭐）

1. **统一战线**
2. **武装斗争**
3. **党的建设**

> 💡 毛泽东 1939 年《〈共产党人〉发刊词》提出

#### 3.3 革命道路

**农村包围城市、武装夺取政权**

#### 3.4 基本经验

- 根本经验：**统一战线和武装斗争**相结合
- 革命对象：帝国主义、封建主义、官僚资本主义
- 革命动力：工人、农民、小资产阶级、民族资产阶级
- 革命领导：无产阶级

### 四、社会主义改造理论

#### 4.1 三大改造

| 对象 | 改造方式 |
|------|----------|
| 农业 | 合作社（互助组→初级社→高级社） |
| 手工业 | 合作社 |
| 资本主义工商业 | **和平赎买**（创举） |

#### 4.2 标志

1956 年底三大改造基本完成，社会主义基本制度在我国初步确立，进入社会主义初级阶段

### 五、毛泽东其他重要思想

#### 5.1 《论十大关系》（1956）

- 标志：探索中国自己的社会主义建设道路的开端
- 基本方针：调动一切积极因素为社会主义事业服务

#### 5.2 正确处理人民内部矛盾（1957）

- 《关于正确处理人民内部矛盾的问题》
- 区分两类不同性质的矛盾：**敌我矛盾**（对抗性）vs **人民内部矛盾**（非对抗性）
- 处理人民内部矛盾的方法：**团结—批评—团结**

## 第四篇：习近平新时代中国特色社会主义思想（核心⭐）

### 一、历史地位

- 2017 年党的十九大确立为党的指导思想
- 是当代中国马克思主义、二十一世纪马克思主义
- 是中华文化和中国精神的时代精华

### 二、"十个明确"（精简版）

1. 明确中国特色社会主义最本质的特征是**中国共产党领导**
2. 明确坚持和发展中国特色社会主义，总任务是实现社会主义现代化和中华民族伟大复兴
3. 明确新时代我国社会主要矛盾是**人民日益增长的美好生活需要和不平衡不充分的发展之间的矛盾**
4. 明确中国特色社会主义事业总体布局是"**五位一体**"、战略布局是"**四个全面**"
5. 明确全面深化改革总目标：完善和发展中国特色社会主义制度，推进国家治理体系和治理能力现代化
6. 明确全面推进依法治国总目标：建设中国特色社会主义法治体系，建设社会主义法治国家
7. 明确必须坚持和完善社会主义基本经济制度
8. 明确党在新时代的强军目标：建设一支听党指挥、能打胜仗、作风优良的人民军队
9. 明确中国特色大国外交要推动构建**人类命运共同体**
10. 明确全面从严治党的战略方针

### 三、"十四个坚持"（必背⭐）

## 第五篇：党史党务

### 易错点提示

1. **物质的唯一特性** = 客观实在性（不是运动）
2. **物质的根本属性** = 运动（不是客观实在性）
3. **实践是认识的基础**（不是来源——来源也是实践，但要分清"基础"和"来源"的表述）
4. **真理的绝对性** ≠ 绝对真理
5. **党的根本性建设** = 政治建设；**基础性建设** = 思想建设
6. **党的宗旨** = 全心全意为人民服务；**党的初心使命** = 为人民谋幸福、为民族谋复兴
7. **新民主主义革命对象** = 帝国主义、封建主义、官僚资本主义（不是"资本主义"）
8. **三大改造完成** = 1956 年底（不是 1949 年）
