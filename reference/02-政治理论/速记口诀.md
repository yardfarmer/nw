# 政治理论 · 速记口诀

---

## 一、马原

### 唯物论三对范畴 → 物意动静时空

```
物质 — 唯一特性客观实在性；根本属性运动
意识 — 人脑机能；客观存在的主观映像
运动 — 绝对
静止 — 相对
时间 — 一维不可逆
空间 — 三维
```

### 物质与意识的关系 → 决定反作用

```
物质决定意识
意识对物质有能动的反作用
方法论：一切从实际出发，实事求是
```

### 两大总特征 → 联系发展

```
联系 — 客观、普遍、多样、条件
发展 — 前进的上升的运动；实质：新事物产生，旧事物灭亡
```

### 三大规律 → 对立统一、量变质变、否定之否定

```
对立统一规律 — 动力和源泉
量变质变规律 — 状态和形式
否定之否定规律 — 方向和道路
```

### 矛盾的基本属性 → 同一斗争

```
同一性 — 相互依存、相互转化
斗争性 — 相互排斥、相互对立
```

### 矛盾普遍性与特殊性 → 无处不在 + 具体分析

```
普遍性 — 矛盾无处不在无时不有
特殊性 — 不同事物矛盾各有特点
方法论：具体问题具体分析
```

### 内因外因关系 → 根据条件

```
内因是变化的根据
外因是变化的条件
外因通过内因起作用
```

### 量变质变 → 必要准备 + 必然结果

```
量变是质变的必要准备
质变是量变的必然结果
质变为新的量变开辟道路
```

### 否定之否定 → 肯定-否定-否定之否定

```
过程：肯定 → 否定 → 否定之否定
特点：螺旋式上升，波浪式前进
方向前进，道路曲折
实质：扬弃（既克服又保留）
```

### 实践是认识的基础 → 源动准目

```
源 — 来源
动 — 动力
准 — 检验真理的唯一标准
目 — 目的
```

### 认识的规律 → 感性到理性，反复无限上升

```
感性认识 — 感觉、知觉、表象（直接具体）
理性认识 — 概念、判断、推理（间接抽象）
认识规律：反复性、无限性、上升性
```

### 真理的属性 → 客绝相具

```
客观性 — 内容客观，标准客观
绝对性 — 包含不依赖人的客观内容
相对性 — 有限、近似、有条件
具体性 — 具体的历史的
```

### 社会存在三要素 → 地人产

```
地理环境
人口因素
物质资料生产方式（决定性）
```

### 社会基本矛盾 → 产经两对

```
生产力 — 生产关系
经济基础 — 上层建筑
```

### 生产力三要素 → 劳动三

```
劳动者（最活跃）
劳动资料（生产工具是标志）
劳动对象
```

### 人民群众三大作用 → 物精变

```
物质财富创造者
精神财富创造者
社会变革决定力量
```

## 五、党史党务

### 党史关键会议 → 一大二大八七古田，遵义七大二中八大

```
一大 1921 — 党的成立
二大 1922 — 反帝反封建纲领
八七会议 1927 — 枪杆子里出政权
古田会议 1929 — 思想建党政治建军
遵义会议 1935 — 生死攸关转折点
七大 1945 — 毛泽东思想确立为指导思想
七届二中全会 1949 — 工作重心转移，两个务必
八大 1956 — 探索社会主义建设良好开端
十一届三中全会 1978 — 改革开放
十九大 2017 — 习思想确立
二十大 2022 — 中国式现代化
二十届三中全会 2024 — 进一步全面深化改革
```

### 党的建设总要求 → 政思组作纪制反

```
政治建设 — 根本性
思想建设 — 基础性
组织建设 — 重要保证
作风建设 — 重要切入点
纪律建设 — 治本之策
制度建设 — 贯穿其中
反腐败斗争 — 必须把权力关进制度笼子
```

### 党的宗旨与初心 → 宗旨服务、初心幸福复兴

```
党的宗旨 — 全心全意为人民服务
党的初心使命 — 为中国人民谋幸福，为中华民族谋复兴
```

### "四风"→ 形官享奢

```
形式主义
官僚主义
享乐主义
奢靡之风
```

### 党的群众路线 → 一切一一切一

```
一切为了群众
一切依靠群众
从群众中来
到群众中去
```

### 七届二中全会"两个务必"→ 谦谨艰奋

```
务必使同志们继续地保持谦虚、谨慎、不骄、不躁的作风
务必使同志们继续地保持艰苦奋斗的作风
```
