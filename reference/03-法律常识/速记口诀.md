# 法律常识 · 速记口诀

---

## 一、宪法

### 宪法特征 → 根本、最高、严程

```
根本法 — 规定国家最根本问题
最高法律效力 — 其他法律的立法依据
制定修改程序严格 — 2/3 全国人大代表通过
```

### 国家机构性质 → 人大权力、政府行政、监委监察、法院审判、检察院监督

```
全国人大 — 最高国家权力机关
国务院 — 最高国家行政机关（总理负责制）
国家监察委员会 — 最高监察机关
最高人民法院 — 最高审判机关
最高人民检察院 — 最高检察机关（法律监督）
中央军委 — 最高军事领导机关（主席负责制）
```

### 修宪程序 → 一五二三

```
提议：1/5 以上全国人大代表 或 全国人大常委会
通过：全国人大全体代表 2/3 以上
```

### 公民权利双重属性 → 劳动 + 受教育

```
劳动权 — 既是权利又是义务
受教育权 — 既是权利又是义务
```

### 选举权三要件 → 中、18、未剥

```
中国公民
年满 18 周岁
未被剥夺政治权利
```

### 民族自治地方 → 区州县（不含民族乡）

```
自治区、自治州、自治县
自治机关首长：实行区域自治的民族的公民担任
```

### 2018 修宪要点 → 习思、监察、主席连任、党的领导

```
确立习近平新时代中国特色社会主义思想指导地位
设立监察委员会
删除国家主席连任两届限制
增加"中国共产党领导是中国特色社会主义最本质的特征"
```

## 二、民法典

### 民法六原则 → 平自公诚守绿

```
平等
自愿
公平
诚信
守法与公序良俗
绿色（节约资源、保护生态环境）
```

### 民事行为能力三分法 → 8 限 18 完

```
无民事行为能力：< 8 周岁
限制民事行为能力：8-18 周岁
完全民事行为能力：≥ 18 周岁（16-18 以劳动收入为主也属完全）
```

### 民事法律行为效力 → 无可待

```
无效 — 欺诈胁迫损害国家利益、恶意串通、违反强制性规定、违背公序良俗
可撤销 — 重大误解、欺诈、胁迫、显失公平
效力待定 — 限制行为能力人实施、无权代理、无权处分
```

### 撤销权期限 → 90 误、1 年欺胁显、5 年最长

```
重大误解：90 日
欺诈、胁迫、显失公平：1 年
最长保护期：5 年
```

### 诉讼时效 → 3 普 20 最

```
普通诉讼时效：3 年
最长保护期：20 年（不适用中止中断）
不适用：抚养费、赡养费、扶养费、物权请求权
```

### 物权三类 → 所、用、担

```
所有权 — 占有、使用、收益、处分
用益物权 — 土地承包经营权、建设用地使用权、宅基地使用权、居住权、地役权
担保物权 — 抵押权、质权、留置权
```

### 善意取得四要件 → 无善价交（不动产登记）

```
出让人无处分权
受让人善意（不知情且无重大过失）
合理价格转让
已交付（动产）或登记（不动产）
```

### 抵质押留区别 → 抵不交、质交、留法定

```
抵押权 — 不交付标的物；不动产抵押必须登记
质权 — 交付标的物
留置权 — 合法占有；法定（不需合意）
```

### 结婚条件 → 男 22 女 20、三代内禁

```
男满 22 周岁
女满 20 周岁
直系血亲和三代以内旁系血亲禁止结婚
```

### 婚姻无效 vs 可撤销 → 无重亲未，可胁隐

```
无效 — 重婚、有禁止结婚亲属关系、未到法定婚龄
可撤销 — 胁迫、被隐瞒重大疾病
```

### 离婚冷静期 → 30+30

```
30 日内任何一方可撤回
30 日届满后 30 日内未亲自申请发证，视为撤回
```

### 诉讼离婚分居 → 2 年

```
因感情不和分居满 2 年，调解无效应准予离婚
```

### 法定继承顺序 → 一配子父，二兄祖外

```
第一顺序：配偶、子女、父母
第二顺序：兄弟姐妹、祖父母、外祖父母
```

### 遗嘱六形式 → 自代打录口公

```
自书遗嘱 — 亲笔书写
代书遗嘱 — 两个以上见证人
打印遗嘱 — 两个以上见证人，每页签名
录音录像遗嘱 — 两个以上见证人
口头遗嘱 — 危急情况
公证遗嘱 — 公证机构办理
```

### 民法典继承新规 → 公证不再优先

```
取消公证遗嘱优先效力
以最后有效遗嘱为准
```

### 遗赠接受期限 → 60 日

```
受遗赠人应在知道受遗赠后 60 日内作出接受表示
到期未表示视为放弃
```

### 侵权归责 → 过错、推定、无过错

```
过错责任 — 一般侵权（谁主张谁举证）
过错推定 — 医疗、建筑物致害（举证责任倒置）
无过错责任 — 高度危险、环境污染、动物致害、产品责任
```

## 三、刑法

### 罪刑法定 → 法无明文不为罪

```
罪刑法定原则
法律面前人人平等
罪责刑相适应
```

### 犯罪构成四要件 → 主体、主观、客体、客观

```
犯罪主体 — 自然人/单位
犯罪主观方面 — 故意（直接/间接）或过失（疏忽/过于自信）
犯罪客体 — 刑法保护的社会关系
犯罪客观方面 — 危害行为、危害结果、因果关系
```

### 刑事责任年龄 → 12·14·16

```
< 12：不负刑事责任
12-14：故意杀人、故意伤害致死或致严重残疾，经最高检核准追诉
14-16：杀伤奸抢、贩火爆投（8 种罪）
≥ 16：完全负刑事责任
```

### 14-16 周岁 8 罪 → 杀伤奸抢、贩火爆投

```
故意杀人
故意伤害致人重伤或死亡
强奸
抢劫
贩卖毒品
放火
爆炸
投放危险物质
```

### 故意 vs 过失 → 希望放任 vs 应见已见

```
直接故意 — 明知会+希望
间接故意 — 明知可能+放任
疏忽大意过失 — 应当预见而未预见
过于自信过失 — 已预见但轻信能避免
```

### 正当防卫 vs 紧急避险 → 人对人不法、险对三方

```
正当防卫 — 不法侵害正在进行；针对不法侵害人本人；无明显超过必要限度
紧急避险 — 危险正在发生；迫不得已；损害较小保护较大
```

### 特殊防卫 → 行杀抢奸绑 + 严重暴力

```
对行凶、杀人、抢劫、强奸、绑架以及其他严重危及人身安全的暴力犯罪
采取防卫造成不法侵害人伤亡的，不属于防卫过当
```

### 犯罪未完成形态 → 预未中

```
预备 — 准备工具制造条件，未着手
未遂 — 已着手，意志以外原因未得逞
中止 — 自动放弃或自动有效防止结果
```

### 中止处罚 → 无损免除、有损减轻

```
没有造成损害 — 应当免除
造成损害 — 应当减轻
```

### 共同犯罪人 → 主从胁教

```
主犯 — 组织领导或起主要作用
从犯 — 次要或辅助作用；从轻、减轻或免除
胁从犯 — 被胁迫；减轻或免除
教唆犯 — 教唆他人犯罪
```

### 主刑 → 管拘有无死

```
管制 — 3 个月-2 年；社区矫正
拘役 — 1-6 个月；每月回家 1-2 天
有期徒刑 — 6 个月-15 年
无期徒刑
死刑 — 不适用于犯罪时未满 18 周岁、审判时怀孕妇女、审判时已满 75 周岁（特别残忍致死除外）
```

### 附加刑 → 罚剥没驱

```
罚金
剥夺政治权利
没收财产
驱逐出境（仅适用于外国人）
```

### 不适用死刑 → 未 18、孕妇、75 特别残忍除外

```
犯罪时不满 18 周岁
审判时怀孕的妇女
审判时已满 75 周岁（但以特别残忍手段致人死亡的除外）
```

### 假释条件 → 有 1/2、无 13、累犯暴 10 不适用

```
有期徒刑：执行原判刑期 1/2 以上
无期徒刑：实际执行 13 年以上
不适用：累犯；故意杀人等暴力犯罪被判处 10 年以上有期徒刑、无期徒刑
```

## 四、行政法

### 行政处罚种类 → 誉财行人

```
声誉罚 — 警告、通报批评
财产罚 — 罚款、没收违法所得、没收非法财物
行为罚 — 暂扣、降级、吊销、限制经营、停产停业、关闭、限制从业
人身自由罚 — 行政拘留
```

### 行政处罚设定权 → 法全、行不人、地不人营

```
法律 — 所有种类
行政法规 — 除限制人身自由外
地方性法规 — 除限制人身自由、吊销营业执照外
规章 — 警告、通报批评、罚款（限额内）
```

### 一事不再罚 → 同一违法不得两次罚款

```
对当事人的同一个违法行为
不得给予两次以上罚款
```

### 当场处罚 → 200 公民、3000 法人

```
违法事实确凿
有法定依据
公民 200 元以下、法人 3000 元以下罚款或警告
```

### 行政处罚追责时效 → 2 年一般、5 年特定

```
一般：2 年
涉及公民生命健康安全、金融安全且有危害后果：5 年
```

### 行政复议 vs 诉讼 → 60 复议、6 月诉讼

```
行政复议：60 日；合法性 + 合理性；一级复议制
行政诉讼：6 个月；原则上只审查合法性；两审终审
```

### 行政强制 → 措施 vs 执行

```
行政强制措施 — 限制人身自由、查封、扣押、冻结
行政强制执行 — 加处罚款、划拨、拍卖、代履行
```

## 五、监察法

### 监察对象 → 六类公职人员

```
1. 党政机关公务员
2. 法律授权或受委托管理公共事务组织中从事公务人员
3. 国有企业管理人员
4. 公办教科文卫体单位中从事管理人员
5. 基层群众性自治组织中从事管理人员
6. 其他依法履行公职人员
```

### 监察措施 → 谈询查冻调封扣搜勘鉴留

```
谈话、询问、查询、冻结、调取、查封、扣押、搜查、勘验检查、鉴定、留置
```

### 留置期限 → 3+3

```
一般 3 个月
特殊情况可延长一次，每次不超过 3 个月
最长 6 个月
```

## 六、劳动法

### 试用期 → 3 月-1 年 1 月；1-3 年 2 月；3 年+或无固定 6 月

```
合同 3 个月-1 年：试用期 1 个月
合同 1-3 年：试用期 2 个月
合同 3 年以上或无固定期限：试用期 6 个月
```

### 试用期工资 → 80% 且不低于最低

```
不低于本单位相同岗位最低档工资或劳动合同约定工资的 80%
并不得低于用人单位所在地最低工资标准
```

### 加班工资 → 1.5 平、2 休、3 节

```
工作日延长：1.5 倍
休息日加班（不能补休）：2 倍
法定节假日：3 倍
```

### 经济补偿 → 满 1 年 1 月，6 月-1 年 1 月，不满 6 月半月

```
每满 1 年：1 个月工资
6 个月以上不满 1 年：1 个月工资
不满 6 个月：半个月工资
高工资（超当地月平均 3 倍）：按 3 倍支付，最长不超过 12 年
```

### 劳动争议解决 → 协调裁诉

```
协商 → 调解 → 仲裁（前置） → 诉讼
仲裁时效：1 年
```

## 七、公务员法

### 公务员三要件 → 公职、编制、财政

```
依法履行公职
纳入国家行政编制
由国家财政负担工资福利
```

### 考核五内容 → 德能勤绩廉（以绩为主）

```
德 — 政治素质、道德品行
能 — 业务能力
勤 — 工作态度
绩 — 工作实绩（核心）
廉 — 廉洁自律
```

### 考核四等级 → 优称基不

```
优秀
称职
基本称职
不称职
```

### 处分六种 → 警记记大降撤开

```
警告 — 6 个月
记过 — 12 个月
记大过 — 18 个月
降级、撤职 — 24 个月
开除
```

### 辞退情形 → 连 2 不称、不胜拒、调拒、不纪不变、旷 15 或 30

```
连续 2 年不称职
不胜任现职不接受其他安排
机构调整拒绝合理安排
不履行义务不守纪律经教育无转变
旷工连续超过 15 天或一年累计超过 30 天
```

### 不得辞退 → 残、病、孕

```
因公致残丧失或部分丧失工作能力
患病或负伤在医疗期内
女性公务员孕期、产假、哺乳期内
```

## 九、法律谚语速记

### 法无授权不可为 vs 法无禁止皆可为

```
法无授权不可为 — 行政法原则（行政机关）
法无禁止皆可为 — 民法原则（公民）
```

### 法不溯及既往 + 从旧兼从轻

```
新法不适用于生效前行为
但有利于当事人的除外（从旧兼从轻）
```

### 疑罪从无

```
刑事诉讼原则
证据不足时应当判决无罪
```

### 一事不再罚

```
对同一违法行为不得给予两次以上罚款
```

### 上诉不加刑

```
二审法院审理上诉案件
不得加重被告人刑罚
```
