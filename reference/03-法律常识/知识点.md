## 第一篇：宪法

### 一、宪法基本理论

#### 1.1 宪法的特征

| 特征 | 内涵 |
|------|------|
| 根本法 | 规定国家最根本、最重要的问题 |
| 最高法律效力 | 宪法是其他法律的立法依据 |
| 制定和修改程序严格 | 修改需全国人大常委会或 1/5 以上全国人大代表提议，全国人大全体代表 2/3 以上通过 |

#### 1.2 宪法基本原则

1. 人民主权原则
2. 基本人权原则
3. 法治原则
4. 民主集中制原则
5. 权力监督与制约原则

### 二、国家基本制度

#### 2.1 国家性质（国体）

**人民民主专政**——工人阶级领导，工农联盟为基础

#### 2.2 政体

**人民代表大会制度**——根本政治制度

#### 2.3 基本经济制度

| 类别 | 内容 |
|------|------|
| 公有制 | 全民所有制 + 集体所有制；公有制经济是社会主义经济制度的基础 |
| 非公有制 | 个体经济、私营经济、外资经济；是社会主义市场经济的重要组成部分 |
| 分配制度 | 按劳分配为主体、多种分配方式并存 |

#### 2.4 国家结构形式

- 单一制
- 行政区划：23 省 + 5 自治区 + 4 直辖市 + 2 特别行政区 = 34 个省级行政区

#### 2.5 民族区域自治制度

- 民族自治地方：自治区、自治州、自治县（不含民族乡）
- 自治机关：自治区、自治州、自治县的人大和政府
- 自治机关首长：自治区主席、自治州州长、自治县县长由实行区域自治的民族的公民担任

#### 2.6 基层群众自治制度

- 村民委员会、居民委员会
- 性质：基层群众性自治组织（不是政权机关）
- 与基层政权关系：乡、镇政府对村委会给予指导、支持和帮助（不是领导）

#### 2.7 特别行政区制度

- 一国两制：社会主义制度（内地）+ 资本主义制度（港澳）
- 高度自治：享有行政管理权、立法权、独立的司法权和终审权
- 中央对特别行政区全面管治权

### 三、公民基本权利与义务（必背⭐）

#### 3.1 基本权利

| 类别 | 权利 |
|------|------|
| **平等权** | 公民在法律面前一律平等 |
| **政治权利与自由** | 选举权和被选举权；言论、出版、集会、结社、游行、示威自由 |
| **人身自由** | 人身自由不受侵犯；人格尊严不受侵犯；住宅不受侵犯；通信自由和通信秘密受法律保护 |
| **宗教信仰自由** | — |
| **监督权** | 批评、建议、申诉、控告、检举权 |
| **社会经济权利** | 财产权；劳动权（**既是权利又是义务**）；休息权；社会保障权 |
| **文化教育权利** | 受教育权（**既是权利又是义务**）；文化活动权 |
| **特定主体权利** | 妇女、儿童、老人、残疾人、华侨等 |

#### 3.2 选举权和被选举权（必背⭐）

三个条件：
1. 中国公民
2. 年满 **18 周岁**
3. 未被剥夺政治权利

#### 3.3 基本义务

1. 维护国家统一和民族团结
2. 遵守宪法和法律
3. 维护国家安全、荣誉和利益
4. 保卫祖国、依法服兵役
5. 依法纳税
6. 受教育（既是权利也是义务）
7. 劳动（既是权利也是义务）
8. 计划生育
9. 父母抚养子女、成年子女赡养父母

### 四、国家机构

#### 4.1 中央国家机构

| 机构 | 性质 | 关键点 |
|------|------|--------|
| **全国人大** | 最高国家权力机关 | 任期 5 年；修改宪法、监督宪法实施 |
| **全国人大常委会** | 全国人大的常设机关 | 解释宪法和法律；21 天召集一次 |
| **国家主席** | 国家代表 | 任期 5 年，无连任限制（2018 修宪后） |
| **国务院** | 最高国家行政机关 | 实行**总理负责制** |
| **中央军委** | 最高军事领导机关 | 实行**主席负责制** |
| **国家监察委员会** | 最高监察机关 | 与中央纪委合署办公 |
| **最高人民法院** | 最高审判机关 | 审判监督 |
| **最高人民检察院** | 最高检察机关 | 法律监督 |

#### 4.2 全国人大 vs 全国人大常委会职权对比

| 职权 | 全国人大 | 全国人大常委会 |
|------|----------|----------------|
| 修宪 | ✔ | ✖ |
| 解释宪法 | ✖ | ✔ |
| 制定基本法律 | ✔ | ✖ |
| 制定非基本法律 | ✖ | ✔ |
| 监督宪法实施 | ✔ | ✔ |
| 选举国家主席 | ✔ | ✖ |
| 决定战争与和平 | ✔ | ✔（战争状态宣布） |

#### 4.3 地方国家机构

- 地方人大：地方国家权力机关（任期 5 年）
- 地方政府：地方国家行政机关
- 地方监委：地方监察机关
- 地方法院：地方审判机关
- 地方检察院：地方检察机关

### 五、2018 年修宪要点

1. 确立**习近平新时代中国特色社会主义思想**在国家政治和社会生活中的指导地位
2. 健全社会主义法治（"健全"改为"建设社会主义法治国家"）
3. 设立**监察委员会**
4. 国家主席任期修改（删除"连续任职不得超过两届"）
5. 增加"中国共产党领导是中国特色社会主义最本质的特征"
6. 增加"国家倡导社会主义核心价值观"

## 第二篇：民法典（核心⭐）

> 2020 年 5 月 28 日通过，2021 年 1 月 1 日施行；共 7 编 1260 条

### 一、民法基本原则

1. 平等原则
2. 自愿原则
3. 公平原则
4. 诚信原则
5. 守法和公序良俗原则
6. 绿色原则（节约资源、保护生态环境）

### 二、民事主体

#### 2.1 自然人

| 类别 | 要点 |
|------|------|
| 民事权利能力 | 始于**出生**，终于**死亡** |
| 民事行为能力 | 分三档（见下） |

##### 民事行为能力三分法（必背⭐）

| 类型 | 条件 | 效力 |
|------|------|------|
| **完全民事行为能力人** | ≥18 周岁；或 16–18 周岁以自己劳动收入为主要生活来源 | 独立实施民事法律行为 |
| **限制民事行为能力人** | 8–18 周岁；或不能完全辨认自己行为的成年人 | 纯获利行为有效；其他需法定代理人追认或同意 |
| **无民事行为能力人** | <8 周岁；或完全不能辨认自己行为的成年人 | 由法定代理人代理 |

##### 监护制度

- 未成年人的监护人：父母（第一顺位）→ 祖父母外祖父母 → 兄姐 → 其他愿意担任且经同意
- 成年人监护人：配偶 → 父母子女 → 其他近亲属 → 其他愿意担任且经同意

#### 2.2 法人

| 类型 | 例子 |
|------|------|
| 营利法人 | 公司 |
| 非营利法人 | 事业单位、社会团体、基金会 |
| 特别法人 | 机关法人、农村集体经济组织、城镇农村合作经济组织、基层群众性自治组织 |

#### 2.3 非法人组织

个人独资企业、合伙企业、不具备法人资格的专业服务机构

### 三、民事法律行为

#### 3.1 民事法律行为效力（必背⭐）

| 类型 | 情形 | 效力 |
|------|------|------|
| **无效** | 欺诈胁迫损害国家利益；恶意串通；违反强制性规定；违背公序良俗 | **自始无效** |
| **可撤销** | 重大误解；欺诈；胁迫；显失公平 | 需当事人请求撤销；撤销前有效 |
| **效力待定** | 限制民事行为能力人实施；无权代理；无权处分 | 需追认 |

#### 3.2 撤销权行使期限

- 重大误解：自知之日起 **90 日**
- 欺诈：自知之日起 **1 年**
- 胁迫：胁迫终止之日起 **1 年**
- 显失公平：自知之日起 **1 年**
- 最长保护期：**5 年**

### 四、代理

#### 4.1 代理种类

- 委托代理
- 法定代理（如父母代理未成年子女）

#### 4.2 无权代理

| 情形 | 处理 |
|------|------|
| 没有代理权 | 被代理人追认则有效；不追认则对被代理人不发生效力 |
| 超越代理权 | 同上 |
| 代理权终止后代理 | 同上 |

#### 4.3 表见代理

行为人没有代理权，但相对人有理由相信其有代理权的，代理行为有效。

### 五、诉讼时效

| 类型 | 期间 |
|------|------|
| 普通诉讼时效 | **3 年** |
| 最长保护期 | **20 年**（不适用中止中断） |
| 不适用诉讼时效 | 支付抚养费、赡养费、扶养费；物权请求权等 |

### 六、物权

#### 6.1 物权种类

| 类别 | 内容 |
|------|------|
| **所有权** | 占有、使用、收益、处分四项权能 |
| **用益物权** | 土地承包经营权、建设用地使用权、宅基地使用权、居住权、地役权 |
| **担保物权** | 抵押权、质权、留置权 |

#### 6.2 善意取得（必背⭐）

条件：
1. 出让人**无处分权**
2. 受让人**善意**（不知情且无重大过失）
3. 以**合理价格**转让
4. 已经**交付**（动产）或**登记**（不动产）

效果：受让人取得所有权；原所有权人有权向无权处分人请求损害赔偿

#### 6.3 共有

| 类型 | 关系 |
|------|------|
| 按份共有 | 按份额享有；份额优先购买权 |
| 共同共有 | 不分份额（如夫妻共同财产、家庭共有） |

#### 6.4 抵押权 vs 质权 vs 留置权

| 类型 | 标的 | 成立方式 |
|------|------|----------|
| **抵押权** | 不动产或动产 | **不交付**标的物；需登记（不动产抵押必须登记） |
| **质权** | 动产或权利 | **交付**标的物 |
| **留置权** | 动产 | 合法占有；法定（不需合意） |

### 七、合同

#### 7.1 合同订立

- **要约**：希望与他人订立合同的意思表示
- **承诺**：受要约人同意要约的意思表示
- 合同成立：承诺生效时

#### 7.2 合同履行原则

1. 全面履行原则
2. 诚信原则
3. 绿色原则

#### 7.3 违约责任

- 继续履行
- 采取补救措施
- 赔偿损失
- 定金罚则
- 违约金

### 八、婚姻家庭（高频🔥）

#### 8.1 结婚条件

| 类型 | 内容 |
|------|------|
| 积极条件 | 男女双方完全自愿；男满 **22 周岁**，女满 **20 周岁** |
| 禁止条件 | 直系血亲和**三代以内旁系血亲**禁止结婚 |

#### 8.2 无效婚姻 vs 可撤销婚姻

| 类型 | 情形 |
|------|------|
| **无效婚姻** | 重婚；有禁止结婚的亲属关系；未到法定婚龄 |
| **可撤销婚姻** | 因胁迫结婚（受胁迫方可在恢复自由之日起 1 年内请求撤销）；被隐瞒重大疾病（自知情之日起 1 年内） |

#### 8.3 夫妻财产制度

##### 法定共同财产

- 工资奖金
- 生产经营收益
- 知识产权收益
- 继承或受赠的财产（遗嘱或赠与合同中确定只归一方的除外）

##### 法定个人财产

- 婚前个人财产
- 因受到人身损害获得的赔偿
- 遗嘱或赠与合同中确定只归一方的财产
- 一方专用的生活用品

##### 约定财产制

可约定婚前和婚后财产归各自所有、共同所有或部分各自所有

#### 8.4 离婚

##### 协议离婚

- 30 天**离婚冷静期**（自婚姻登记机关收到申请之日起 30 日内，任何一方不愿离婚的，可撤回）
- 30 天后再 30 天内未亲自申请发证的，视为撤回

##### 诉讼离婚

法院调解无效，应准予离婚的情形：

1. 重婚或与他人同居
2. 实施家庭暴力或虐待、遗弃家庭成员
3. 有赌博、吸毒等恶习屡教不改
4. 因感情不和分居满 **2 年**
5. 其他导致夫妻感情破裂的情形

### 九、继承（必背⭐）

#### 9.1 法定继承顺序

| 顺序 | 继承人 |
|------|--------|
| **第一顺序** | 配偶、子女、父母 |
| **第二顺序** | 兄弟姐妹、祖父母、外祖父母 |

> 丧偶儿媳/女婿对公婆/岳父母尽了主要赡养义务的，作为第一顺序继承人

#### 9.2 遗嘱形式

| 形式 | 要求 |
|------|------|
| 自书遗嘱 | 亲笔书写，签名，注明年月日 |
| 代书遗嘱 | 两个以上无利害关系见证人在场见证 |
| 打印遗嘱 | 两个以上见证人在场见证，遗嘱人和见证人在每一页签名 |
| 录音录像遗嘱 | 两个以上见证人在场见证 |
| 口头遗嘱 | **危急情况**；危急情况消除后能书面立遗嘱的，口头遗嘱无效 |
| 公证遗嘱 | 经公证机构办理 |

> ⚠️ 民法典新规定：取消公证遗嘱优先效力，**以最后有效遗嘱为准**

#### 9.3 代位继承

被继承人的子女先于被继承人死亡的，由该子女的**直系晚辈血亲**代位继承。

#### 9.4 遗赠

- 遗赠是给法定继承人以外的人
- 受遗赠人应在知道受遗赠后 **60 日**内作出接受表示，到期未表示视为放弃

### 十、侵权责任

#### 10.1 归责原则

| 原则 | 适用情形 |
|------|----------|
| **过错责任原则** | 一般侵权行为（谁主张谁举证） |
| **过错推定原则** | 医疗损害、建筑物致害等（举证责任倒置） |
| **无过错责任原则** | 高度危险作业、环境污染、饲养动物致害等 |

#### 10.2 特殊侵权（必背⭐）

| 类型 | 归责原则 |
|------|----------|
| 饲养动物致害 | 无过错责任 |
| 高度危险作业 | 无过错责任 |
| 环境污染 | 无过错责任 |
| 产品责任 | 无过错责任（生产者） |
| 交通事故 | 过错责任（机动车间）；过错推定（机动车对行人） |
| 校园伤害 | 过错责任/过错推定 |
| 雇员致害 | 用人单位承担（无过错责任） |
| 建筑物物件致害 | 过错推定 |

## 第三篇：刑法

### 一、刑法基本原则

1. **罪刑法定原则**：法无明文规定不为罪，法无明文规定不处罚
2. **法律面前人人平等原则**
3. **罪责刑相适应原则**

### 二、犯罪构成要件

| 要件 | 内容 |
|------|------|
| 犯罪主体 | 实施犯罪的人（自然人/单位） |
| 犯罪主观方面 | 故意（直接/间接）或过失（疏忽大意/过于自信） |
| 犯罪客体 | 刑法所保护而为犯罪所侵害的社会关系 |
| 犯罪客观方面 | 危害行为、危害结果、因果关系 |

### 三、刑事责任年龄（必背⭐）

| 年龄段 | 刑事责任 |
|--------|----------|
| **<12 周岁** | 不负刑事责任 |
| **12–14 周岁** | 犯故意杀人、故意伤害罪，致人死亡或特别残忍手段致人重伤造成严重残疾，**经最高检核准追诉**的，负刑事责任 |
| **14–16 周岁** | 犯故意杀人、故意伤害致人重伤或死亡、强奸、抢劫、贩卖毒品、放火、爆炸、投放危险物质罪的，负刑事责任（8 种罪） |
| **≥16 周岁** | 完全负刑事责任 |

> 💡 14–16 周岁记忆口诀：**杀伤奸抢、贩火爆投**

### 四、犯罪主观方面

#### 4.1 故意

| 类型 | 认知 | 意志 |
|------|------|------|
| **直接故意** | 明知危害结果会发生 | 希望 |
| **间接故意** | 明知危害结果可能发生 | 放任 |

#### 4.2 过失

| 类型 | 认知 | 意志 |
|------|------|------|
| **疏忽大意过失** | 应当预见而未预见 | 不希望 |
| **过于自信过失** | 已预见但轻信能避免 | 不希望 |

### 五、正当防卫与紧急避险（必背⭐）

| 概念 | 条件 |
|------|------|
| **正当防卫** | 1. 不法侵害**正在进行**；2. 针对不法侵害人**本人**；3. 没有明显超过必要限度；4. 出于防卫意图 |
| **紧急避险** | 1. 危险**正在发生**；2. **迫不得已**；3. 损害较小合法权益保护较大合法权益；4. 出于避险意图 |

#### 区别要点

| 维度 | 正当防卫 | 紧急避险 |
|------|----------|----------|
| 危险来源 | 人的不法侵害 | 自然力、动物、人的侵害等 |
| 针对对象 | 不法侵害人本人 | 第三方合法权益 |
| 限度要求 | 不能明显超过必要限度 | 损害小于保护利益 |
| 限度过当 | 防卫过当 | 避险过当 |

#### 特殊防卫

对正在进行**行凶、杀人、抢劫、强奸、绑架以及其他严重危及人身安全的暴力犯罪**采取防卫行为，造成不法侵害人伤亡的，**不属于防卫过当**。

### 六、犯罪的未完成形态

| 形态 | 条件 | 处罚 |
|------|------|------|
| 犯罪预备 | 为犯罪准备工具、制造条件；**未着手** | 比照既遂从轻、减轻或免除 |
| 犯罪未遂 | 已着手；**意志以外原因**未得逞 | 比照既遂从轻或减轻 |
| 犯罪中止 | **自动**放弃犯罪或自动有效防止结果发生 | 没造成损害**应当免除**；造成损害**应当减轻** |

### 七、共同犯罪

#### 7.1 共同犯罪人种类

| 角色 | 处罚 |
|------|------|
| **主犯** | 组织、领导犯罪集团进行犯罪活动的或在共同犯罪中起主要作用的 |
| **从犯** | 起次要或辅助作用；应当从轻、减轻或免除处罚 |
| **胁从犯** | 被胁迫参加；应当减轻或免除处罚 |
| **教唆犯** | 教唆他人犯罪；按其在共同犯罪中的作用处罚 |

### 八、刑罚

#### 8.1 主刑

管制 → 拘役 → 有期徒刑 → 无期徒刑 → 死刑

| 主刑 | 期限 | 关键点 |
|------|------|--------|
| 管制 | 3 个月–2 年（数罪并罚最高 3 年） | 不羁押；社区矫正 |
| 拘役 | 1–6 个月（数罪并罚最高 1 年） | 公安机关就近执行；每月可回家 1–2 天 |
| 有期徒刑 | 6 个月–15 年（数罪并罚最高 25 年） | 监狱执行 |
| 无期徒刑 | — | 监狱执行 |
| 死刑 | — | 不适用于：犯罪时不满 18 周岁的人；审判时怀孕的妇女；审判时已满 75 周岁的人（但以特别残忍手段致人死亡的除外） |

#### 8.2 附加刑

| 附加刑 | 说明 |
|--------|------|
| 罚金 | — |
| 剥夺政治权利 | 选举权和被选举权；言论出版集会结社游行示威自由；担任国家机关职务；担任国有公司企业事业单位和人民团体领导职务 |
| 没收财产 | 没收犯罪分子个人所有财产的一部或全部 |
| 驱逐出境 | 仅适用于外国人 |

### 九、刑罚执行

#### 9.1 减刑条件

- 适用对象：被判处管制、拘役、有期徒刑、无期徒刑的犯罪分子
- 实质条件：在执行期间，认真遵守监规、接受教育改造，确有悔改表现或立功表现

#### 9.2 假释条件

- 适用对象：被判处有期徒刑（执行原判刑期 1/2 以上）、无期徒刑（实际执行 13 年以上）的犯罪分子
- 实质条件：认真遵守监规、接受教育改造，确有悔改表现，没有再犯罪危险
- 不适用：累犯；因故意杀人、强奸、抢劫、绑架、放火、爆炸、投放危险物质或者有组织的暴力性犯罪被判处 10 年以上有期徒刑、无期徒刑的

## 第四篇：行政法

### 一、行政行为分类

| 类型 | 说明 |
|------|------|
| **抽象行政行为** | 制定法规、规章等具有普遍约束力的行为 |
| **具体行政行为** | 针对特定对象的具体处理决定 |

### 二、行政处罚（必背⭐）

#### 2.1 处罚种类

| 类别 | 内容 |
|------|------|
| **声誉罚** | 警告、通报批评 |
| **财产罚** | 罚款、没收违法所得、没收非法财物 |
| **行为罚** | 暂扣许可证件、降低资质等级、吊销许可证件、限制开展生产经营活动、责令停产停业、责令关闭、限制从业 |
| **人身自由罚** | 行政拘留 |

#### 2.2 行政处罚设定权

| 机关 | 可设定的处罚 |
|------|-------------|
| **法律** | 所有种类 |
| **行政法规** | 除限制人身自由外的处罚 |
| **地方性法规** | 除限制人身自由、吊销营业执照外的处罚 |
| **部门规章** | 警告、通报批评、罚款（一定限额内） |
| **地方政府规章** | 警告、通报批评、罚款（一定限额内） |

#### 2.3 一事不再罚原则

对当事人的同一个违法行为，**不得给予两次以上罚款**的行政处罚。

#### 2.4 当场处罚（简易程序）

- 适用：违法事实确凿；有法定依据；对公民处以 200 元以下、对法人或者其他组织处以 3000 元以下罚款或者警告的行政处罚
- 程序：出示执法证件 → 填写预定格式编有号码的行政处罚决定书 → 当场交付当事人

#### 2.5 行政处罚追责时效

- 一般：**2 年**
- 涉及公民生命健康安全、金融安全且有危害后果：**5 年**

### 三、行政许可

#### 3.1 设定

| 机关 | 设定权限 |
|------|----------|
| 法律 | 可以设定任何事项的行政许可 |
| 行政法规 | 在尚未制定法律的情况下设定 |
| 国务院决定 | 在尚未制定法律、行政法规的情况下设定（临时性） |
| 地方性法规 | 在尚未制定法律、行政法规的情况下设定 |
| 省级政府规章 | 临时性设定（1 年内未上升为地方性法规的，应停止） |

#### 3.2 行政许可原则

许可法定、公开公平公正、便民高效、信赖保护

### 四、行政强制

| 类别 | 内容 |
|------|------|
| 行政强制措施 | 限制人身自由；查封场所、设施或财物；扣押财物；冻结存款汇款 |
| 行政强制执行 | 加处罚款；划拨存款汇款；拍卖查封扣押财物；代履行；申请法院强制执行 |

### 五、行政复议与行政诉讼

#### 5.1 行政复议

- 申请期限：自知道具体行政行为之日起 **60 日**
- 受理机关：上一级行政机关或本级人民政府
- 审查范围：**合法性 + 合理性**
- 决定类型：维持、撤销、变更、确认违法、责令履行

#### 5.2 行政诉讼

- 起诉期限：**6 个月**（自知道或应当知道行政行为之日起）
- 审查范围：**合法性**（合理性一般不审查，但显失公平的可变更）
- 审理程序：一审、二审（两审终审）

#### 5.3 行政复议 vs 行政诉讼对比

| 维度 | 行政复议 | 行政诉讼 |
|------|----------|----------|
| 性质 | 行政救济 | 司法救济 |
| 受理机关 | 上级行政机关 | 人民法院 |
| 审查范围 | 合法性 + 合理性 | 原则上只审查合法性 |
| 时效 | 60 日 | 6 个月 |
| 程序 | 一级复议制 | 两审终审 |
| 是否收费 | 不收费 | 按规定收费 |

#### 5.4 复议前置情形

1. 自然资源权属确认
2. 法律法规规定的其他情形（如纳税争议、商标专利复审等）

## 第五篇：监察法

### 一、监察机关性质

- 国家监察机关
- 行使国家监察权
- 与党的纪律检查委员会**合署办公**

### 二、监察对象

所有行使公权力的公职人员，包括：

1. 中国共产党机关、人大机关、行政机关、政协机关、监察机关、审判机关、检察机关、民主党派机关和工商联机关的公务员等
2. 法律授权或者受委托管理公共事务的组织中从事公务的人员
3. 国有企业管理人员
4. 公办的教育、科研、文化、医疗卫生、体育等单位中从事管理的人员
5. 基层群众性自治组织中从事管理的人员
6. 其他依法履行公职的人员

### 三、监察措施

谈话、询问、查询、冻结、调取、查封、扣押、搜查、勘验检查、鉴定、**留置**

### 四、留置（必背⭐）

| 维度 | 内容 |
|------|------|
| 适用条件 | 涉嫌严重违法犯罪的；涉及案情重大复杂的；可能逃跑自杀的；可能串供伪造证据的等 |
| 批准 | 监察机关领导人员集体研究决定 |
| 最长期限 | **3 个月**（特殊情况可延长一次，每次不超过 3 个月）——实际最长不超过 **6 个月** |

### 五、监察原则

监察委员会**独立行使监察权**，不受行政机关、社会团体和个人的干涉。

## 第六篇：劳动法与劳动合同法

### 一、劳动合同

#### 1.1 劳动合同期限

| 类型 | 试用期上限 |
|------|------------|
| 3 个月以上不满 1 年 | 1 个月 |
| 1 年以上不满 3 年 | 2 个月 |
| 3 年以上或无固定期限 | **6 个月** |

> ⚠️ 同一用人单位与同一劳动者只能约定**一次试用期**

#### 1.2 试用期工资

不得低于本单位相同岗位最低档工资或劳动合同约定工资的 **80%**，并不得低于用人单位所在地的最低工资标准。

### 二、工作时间与加班

#### 2.1 标准工时

- 每日工作 8 小时
- 每周工作 40 小时（5 天）

#### 2.2 加班工资（必背⭐）

| 时间 | 加班工资 |
|------|----------|
| 工作日延长 | 1.5 倍 |
| 休息日加班（不能补休） | 2 倍 |
| 法定节假日 | **3 倍** |

### 三、经济补偿

| 工作年限 | 经济补偿 |
|----------|----------|
| 每满 1 年 | 1 个月工资 |
| 6 个月以上不满 1 年 | 1 个月工资 |
| 不满 6 个月 | 半个月工资 |

> 月工资指劳动者在劳动合同解除或终止前 12 个月的平均工资；高于本地区上年度职工月平均工资 3 倍的，按 3 倍支付，且最长不超过 12 年。

### 四、劳动争议解决

协商 → 调解 → 仲裁 → 诉讼

- 劳动争议仲裁前置：必须先经过仲裁才能提起诉讼
- 仲裁时效：1 年

## 第七篇：公务员法

### 一、公务员范围

依法履行公职、纳入**国家行政编制**、由国家财政负担工资福利的工作人员。

### 二、公务员条件

1. 具有中华人民共和国国籍
2. 年满 18 周岁
3. 拥护中华人民共和国宪法
4. 具有良好的品行
5. 具有正常履行职责的身体条件
6. 具有符合职位要求的文化程度和工作能力
7. 法律规定的其他条件

### 三、考核

#### 3.1 考核内容

德、能、勤、绩、廉，**以绩为主**

#### 3.2 考核结果

| 等级 | 处理 |
|------|------|
| 优秀 | 称职以上 |
| 称职 | 正常晋级增资 |
| 基本称职 | 不得晋升 |
| 不称职 | 降低一个职务层次任职 |

### 四、处分（必背⭐）

#### 4.1 处分种类

警告 → 记过 → 记大过 → 降级 → 撤职 → **开除**

#### 4.2 处分期间

| 处分 | 期间 |
|------|------|
| 警告 | 6 个月 |
| 记过 | 12 个月 |
| 记大过 | 18 个月 |
| 降级、撤职 | 24 个月 |

#### 4.3 处分后果

- 受处分期间不得晋升职务、职级和级别
- 受警告处分的，可以晋升工资档次
- 受记过、记大过、降级、撤职处分的，不得晋升工资档次
- 受撤职处分的，按规定降低级别

### 五、辞退

#### 5.1 辞退情形

1. 在年度考核中，连续 2 年被确定为不称职的
2. 不胜任现职工作，又不接受其他安排的
3. 因所在机关调整、撤销、合并或者缩减编制员额需要调整工作，本人拒绝合理安排的
4. 不履行公务员义务，不遵守法律和公务员纪律，经教育仍无转变，不适合继续在机关工作，又不宜给予开除处分的
5. 旷工或者因公外出、请假期满无正当理由逾期不归连续超过 15 天，或一年内累计超过 30 天的

#### 5.2 不得辞退情形

1. 因公致残，被确认丧失或者部分丧失工作能力的
2. 患病或者负伤，在规定的医疗期内的
3. 女性公务员在孕期、产假、哺乳期内的
4. 法律、行政法规规定的其他不得辞退的情形

## 常考法律谚语与原则

| 谚语/原则 | 法律含义 |
|-----------|----------|
| 法无授权不可为 | 行政法原则，行政机关必须在法律授权范围内行事 |
| 法无禁止皆可为 | 民法原则，法律未禁止的行为公民均可实施 |
| 法不溯及既往 | 新法不适用于其生效前发生的行为（从旧兼从轻例外） |
| 疑罪从无 | 刑事诉讼原则，证据不足时应当判决无罪 |
| 谁主张谁举证 | 一般举证规则 |
| 举证责任倒置 | 特定案件中由被告举证（如医疗损害、环境污染等） |
| 一事不再罚 | 对同一违法行为不得给予两次以上罚款 |
| 罪刑法定 | 法无明文规定不为罪 |
| 上诉不加刑 | 二审法院审理上诉案件，不得加重被告人刑罚 |

## 备考提示

### 必背 TOP 10

1. 民事行为能力三分法（8/18 周岁）
2. 刑事责任年龄四档（12/14/16 周岁）
3. 行政处罚种类与设定权
4. 行政复议 vs 行政诉讼（时效 60 日 vs 6 个月）
5. 善意取得四要件
6. 离婚冷静期 30 天
7. 法定继承两顺序
8. 公务员处分六种
9. 加班工资倍数（1.5/2/3 倍）
10. 国家机构性质

### 易错点

1. **8 周岁**开始具有限制民事行为能力（不是 10 周岁）
2. **12 周岁**开始可能负刑事责任（特定罪名+最高检核准）
3. **离婚冷静期是 30 天**（不是 90 天）
4. **诉讼时效 3 年**（不是 2 年，民法典已修改）
5. **公证遗嘱不再优先**（民法典新规定）
6. **行政复议 60 日，行政诉讼 6 个月**（不要混淆）
7. **国务院实行总理负责制**（不是委员会制）
8. **国家主席无连任限制**（2018 修宪删除"连续任职不得超过两届"）
9. **未成年人不适用死刑**（犯罪时不满 18 周岁）
10. **监察留置最长 6 个月**（3+3）
