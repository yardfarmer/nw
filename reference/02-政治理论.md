## 第一篇：马克思主义基本原理（马原）

### 一、唯物论

| 概念 | 要点 |
|------|------|
| 物质 | 物质的唯一特性是**客观实在性** |
| 意识 | 意识是物质世界长期发展的产物，是人脑的机能，是客观存在的主观映像 |
| 关系 | 物质决定意识，意识对物质有能动的反作用 |

### 二、唯物辩证法

#### 两大总特征

1. **联系**：事物内部各要素之间和事物之间相互影响、相互制约、相互作用的关系
2. **发展**：前进的、上升的运动，实质是**新事物的产生和旧事物的灭亡**

#### 三大规律（必背⭐）

| 规律 | 核心内容 |
|------|----------|
| **对立统一规律** | 矛盾是事物发展的根本动力；矛盾的基本属性是同一性和斗争性；内因是变化的根据，外因是变化的条件 |
| **量变质变规律** | 量变是质变的必要准备，质变是量变的必然结果；质变为新的量变开辟道路 |
| **否定之否定规律** | 事物发展是螺旋式上升、波浪式前进；方向是前进的，道路是曲折的 |

#### 矛盾特殊性（高频考点🔥）

- 不同事物的矛盾各有其特点
- 同一事物的矛盾在不同发展阶段各有不同特点
- 构成事物的诸多矛盾及每一矛盾的不同方面各有不同的性质、地位和作用
- **方法论**：具体问题具体分析

### 三、认识论

| 概念 | 要点 |
|------|------|
| 实践 | 实践是认识的基础（来源、动力、标准、目的） |
| 认识 | 认识具有反复性、无限性、上升性 |
| 真理 | 真理的客观性：真理的内容是客观的；检验真理的标准是客观的（实践） |

### 四、唯物史观

| 概念 | 要点 |
|------|------|
| 社会存在 | 社会物质生活条件的总和（地理环境、人口因素、物质资料生产方式） |
| 社会意识 | 社会精神生活的总和；社会存在决定社会意识，社会意识反作用于社会存在 |
| 人民群众 | 人民群众是历史的创造者（物质财富、精神财富、社会变革的决定力量） |

## 第二篇：毛泽东思想

### 一、毛泽东思想活的灵魂（必背⭐）

1. **实事求是** — 根本出发点和方法论
2. **群众路线** — 根本工作路线（一切为了群众，一切依靠群众，从群众中来，到群众中去）
3. **独立自主** — 根本立场

### 二、新民主主义革命理论

| 要素 | 内容 |
|------|------|
| 总路线 | 无产阶级领导的，人民大众的，反对帝国主义、封建主义和官僚资本主义的革命 |
| 三大法宝 | 统一战线、武装斗争、党的建设 |
| 革命道路 | 农村包围城市、武装夺取政权 |
| 根本经验 | 统一战线和武装斗争相结合 |

### 三、社会主义改造理论

- **三大改造**：农业、手工业、资本主义工商业
- 对资本主义工商业的改造：**和平赎买**政策
- 标志：1956 年底三大改造基本完成，社会主义基本制度确立

## 第四篇：习近平新时代中国特色社会主义思想（核心重点🔥）

### "十个明确"（精简版）

1. 明确中国特色社会主义最本质的特征是**中国共产党领导**
2. 明确坚持和发展中国特色社会主义，总任务是实现社会主义现代化和中华民族伟大复兴
3. 明确新时代我国社会主要矛盾是**人民日益增长的美好生活需要和不平衡不充分的发展之间的矛盾**
4. 明确中国特色社会主义事业总体布局是"五位一体"、战略布局是"四个全面"
5. 明确全面深化改革总目标
6. 明确全面推进依法治国总目标
7. 明确必须坚持和完善社会主义基本经济制度
8. 明确党在新时代的强军目标
9. 明确中国特色大国外交要推动构建人类命运共同体
10. 明确全面从严治党的战略方针

### "十四个坚持"（重点）

1. 坚持党对一切工作的领导
2. 坚持以人民为中心
3. 坚持全面深化改革
4. 坚持新发展理念
5. 坚持人民当家作主
6. 坚持全面依法治国
7. 坚持社会主义核心价值体系
8. 坚持在发展中保障和改善民生
9. 坚持人与自然和谐共生
10. 坚持总体国家安全观
11. 坚持党对人民军队的绝对领导
12. 坚持"一国两制"和推进祖国统一
13. 坚持推动构建人类命运共同体
14. 坚持全面从严治党

### 新发展理念（必背⭐）

创新、协调、绿色、开放、共享

### "五位一体"总体布局

经济建设、政治建设、文化建设、社会建设、生态文明建设

### "四个全面"战略布局

全面建设社会主义现代化国家、全面深化改革、全面依法治国、全面从严治党

## 第五篇：党史党务

### 重要会议

| 会议 | 时间 | 意义 |
|------|------|------|
| 一大 | 1921 年 | 中国共产党成立 |
| 遵义会议 | 1935 年 | 事实上确立了毛泽东在党中央的领导地位，生死攸关的转折点 |
| 七大 | 1945 年 | 确立毛泽东思想为指导思想 |
| 十一届三中全会 | 1978 年 | 开启改革开放，工作重心转移到经济建设 |
| 十八届三中全会 | 2013 年 | 全面深化改革 |
| 十九大 | 2017 年 | 确立习近平新时代中国特色社会主义思想 |
| 二十大 | 2022 年 | 以中国式现代化全面推进中华民族伟大复兴 |

### 党的建设

- **政治建设**：党的根本性建设
- **思想建设**：党的基础性建设
- **组织建设**：重要保证
- **作风建设**：重要切入点
- **纪律建设**：治本之策
- **制度建设**：贯穿其中
