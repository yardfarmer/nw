# 公文知识

> 考试权重：约 10% | 考点固定，容易拿分，不可丢

## 一、15 种法定公文

| 类别 | 文种 | 用途 |
|------|------|------|
| 领导指导性 | 命令（令）、决定、决议、批复 | 上级对下级 |
| 呈报性 | 报告、请示 | 下级对上级 |
| 公布性 | 公告、通告 | 向社会公众 |
| 商洽性 | 函 | 不相隶属机关之间 |
| 会议性 | 纪要 | 记录会议情况 |
| 通用性 | 通知、通报、意见、议案 | 多种用途 |

## 二、高频易混文种辨析（必背⭐）

### 公告 vs 通告

| 对比 | 公告 | 通告 |
|------|------|------|
| 对象 | 面向**国内外** | 面向**辖区内部** |
| 发布机关 | 高级别机关 | 各级行政机关 |
| 内容 | 重大事项 | 应遵守或周知事项 |

### 通知 vs 通报

| 对比 | 通知 | 通报 |
|------|------|------|
| 功能 | 告知**事项**，要求执行 | **表彰/批评/传达**，起教育作用 |
| 事态 | 事前告知 | 事后通报 |

### 报告 vs 请示

| 对比 | 报告 | 请示 |
|------|------|------|
| 目的 | 汇报工作、反映情况 | 请求批准/指示 |
| 是否要求批复 | **否** | **是** |
| 行文时机 | 事后或事中 | **事前** |
| 是否一文一事 | 否 | **是** |

### 请示 vs 函

| 对比 | 请示 | 函 |
|------|------|------|
| 关系 | **上行文**（下级对上级） | **平行文**（不相隶属机关） |
| 功能 | 请求批准 | 商洽工作、询问/答复问题 |

## 三、公文格式

### 版头

| 要素 | 说明 |
|------|------|
| 份号 | 公文印制份数的顺序号（涉密公文标注） |
| 密级 | 绝密★、机密★、秘密★ |
| 紧急程度 | 特急★、加急★ |
| 发文机关标志 | 红色大字 |
| 发文字号 | 机关代字+年份+序号，如"国发〔2025〕1号" |
| 签发人 | 上行文标注签发人姓名 |

### 主体

| 要素 | 说明 |
|------|------|
| 标题 | 发文机关+事由+文种 |
| 主送机关 | 受理公文的机关 |
| 正文 | 公文核心内容 |
| 附件说明 | 附件序号+名称 |
| 发文机关署名 | 发文机关名称 |
| 成文日期 | 用**阿拉伯数字**，如 2025年3月5日 |
| 印章 | 除纪要和有特定标志外，均加盖印章 |

### 版记

| 要素 | 说明 |
|------|------|
| 抄送机关 | 除主送机关外需要执行或知晓的其他机关 |
| 印发机关 | 印发部门 |
| 印发日期 | 与发文日期可以不同 |

## 四、行文规则（高频🔥）

1. **上行文**：原则上主送一个上级机关，根据需要同时抄送相关上级机关和同级机关，**不抄送下级机关**
2. **下行文**：主送受理机关，根据需要抄送相关机关，**重要行文应同时抄送发文机关的直接上级机关**
3. **联合行文**：**同级机关**可以联合行文
4. **请示规则**：
   - 一文一事
   - 不得抄送下级机关
   - 不越级请示
   - 不得在报告中夹带请示事项
5. **意见**：可作上行文、下行文、平行文使用（最灵活）

## 五、公文处理

### 发文办理

拟稿 → 审核 → 签发 → 印制 → 发出

### 收文办理

签收 → 登记 → 初审 → 承办 → 传阅 → 催办 → 答复

## 六、常考细节

| 考点 | 答案 |
|------|------|
| 成文日期格式 | 阿拉伯数字 |
| 请示能否夹带在报告中 | 不能 |
| 联合行文条件 | 同级机关 |
| 意见的行文方向 | 上行/下行/平行均可 |
| 涉密公文份号 | 必须标注 |
| 发文字号中年份 | 用六角括号〔〕 |
