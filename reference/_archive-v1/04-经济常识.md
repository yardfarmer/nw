# 经济常识

> 考试权重：约 10-15% | 侧重宏观经济政策，常与时政结合出题

---

## 第一篇：微观经济学

### 一、供求与价格

| 概念 | 要点 |
|------|------|
| 需求定律 | 价格上升，需求量下降（反向变动） |
| 供给定律 | 价格上升，供给量增加（同向变动） |
| 均衡价格 | 供给量 = 需求量时的价格 |
| 弹性 | 需求价格弹性：需求量变动率 / 价格变动率 |

### 二、市场类型

| 类型 | 厂商数量 | 产品 | 进入壁垒 |
|------|----------|------|----------|
| 完全竞争 | 众多 | 同质 | 无 |
| 垄断竞争 | 较多 | 差异化 | 低 |
| 寡头垄断 | 少数 | 同质或差异化 | 高 |
| 完全垄断 | 一个 | 唯一 | 极高 |

### 三、市场失灵

| 原因 | 说明 |
|------|------|
| 外部性 | 正外部性（教育）和负外部性（污染） |
| 公共物品 | 非排他性 + 非竞争性 |
| 信息不对称 | 逆向选择（事前）、道德风险（事后） |
| 垄断 | 抑制竞争，损害效率 |

## 第二篇：宏观经济学（高频🔥）

### 一、核心指标

| 指标 | 含义 |
|------|------|
| GDP | 国内生产总值——一国境内一定时期生产的最终产品和服务的市场价值 |
| GNP | 国民生产总值——一国国民一定时期生产的最终产品和服务的市场价值 |
| CPI | 居民消费价格指数——衡量通货膨胀的主要指标 |
| 失业率 | 失业人口 / 劳动力人口 |

### 二、通货膨胀与通货紧缩

| 情形 | 表现 | 对策 |
|------|------|------|
| 通货膨胀 | 物价持续上涨、货币贬值 | 紧缩性财政政策 + 紧缩性货币政策 |
| 通货紧缩 | 物价持续下跌、需求不足 | 扩张性财政政策 + 扩张性货币政策 |

### 三、财政政策

| 工具 | 内容 |
|------|------|
| 税收 | 增税→紧缩，减税→扩张 |
| 国债 | 发行国债→扩张（增加政府支出） |
| 政府购买 | 增加购买→扩张 |
| 转移支付 | 增加转移支付→扩张 |

### 四、货币政策

| 工具 | 内容 |
|------|------|
| 法定存款准备金率 | 上调→紧缩（银行可贷资金减少） |
| 再贴现率 | 上调→紧缩（商业银行借款成本增加） |
| 公开市场操作 | 卖出债券→紧缩（回收流动性） |
| 基准利率 | 上调→紧缩（借贷成本增加） |

> 口诀：**"买债降准降利率"= 扩张；"卖债升准升利率"= 紧缩**

### 五、汇率

| 变动 | 影响 |
|------|------|
| 本币升值 | 利于进口，不利于出口；利于出国旅游/留学 |
| 本币贬值 | 利于出口，不利于进口；利于吸引外资 |

## 第三篇：中国经济热点

### 一、新发展理念（必背⭐）

创新、协调、绿色、开放、共享

### 二、新发展格局

以国内大循环为主体、国内国际双循环相互促进

### 三、高质量发展

- 从"有没有"转向"好不好"
- 核心是提高全要素生产率

### 四、共同富裕

- 不是同时富裕，也不是同等富裕
- 不是劫富济贫，而是先富带后富
- 浙江是**高质量发展建设共同富裕示范区**

### 五、新质生产力

- 以科技创新为主导
- 高科技、高效能、高质量
- 绿色发展是底色

### 六、供给侧结构性改革

- 去产能、去库存、去杠杆、降成本、补短板
- 从供给侧发力，提高供给体系质量

## 第四篇：马克思主义政治经济学

| 概念 | 要点 |
|------|------|
| 商品 | 使用价值（自然属性）+ 价值（社会属性） |
| 使用价值 | 满足人的需要的有用性 |
| 价值 | 凝结在商品中的无差别的人类劳动 |
| 价值规律 | 商品价值量由社会必要劳动时间决定；商品交换以价值量为基础 |
| 剩余价值 | 工人创造的超过劳动力价值的那部分价值，被资本家无偿占有 |

### 劳动价值论深化

| 概念 | 要点 |
|------|------|
| 具体劳动 | 创造使用价值（如裁缝做衣服） |
| 抽象劳动 | 创造价值（无差别的人类劳动耗费） |
| 社会必要劳动时间 | 在现有社会正常生产条件下，在社会平均劳动熟练程度下制造某种使用价值所需的劳动时间 |
| 劳动生产率 | 与单位商品价值量成反比，与同一劳动同一时间创造的价值总量无关 |

### 资本与剩余价值

| 概念 | 要点 |
|------|------|
| 不变资本（c） | 购买生产资料的资本，不发生价值增值 |
| 可变资本（v） | 购买劳动力的资本，是剩余价值的源泉 |
| 剩余价值率 | m' = m/v，反映剥削程度 |
| 绝对剩余价值 | 延长工作日、提高劳动强度 |
| 相对剩余价值 | 提高劳动生产率，缩短必要劳动时间 |

## 第五篇：国际贸易与金融

### 一、国际贸易基础

| 概念 | 要点 |
|------|------|
| 比较优势理论 | 李嘉图提出，各国应生产并出口比较优势产品 |
| 贸易顺差 | 出口 > 进口 |
| 贸易逆差 | 进口 > 出口 |
| 关税 | 进口关税保护国内产业 |
| 贸易壁垒 | 关税壁垒 + 非关税壁垒（配额、技术壁垒、反倾销等） |

### 二、国际经济组织（必背⭐）

| 组织 | 要点 |
|------|------|
| WTO（世界贸易组织） | 促进贸易自由化，解决贸易争端；中国 2001 年加入 |
| IMF（国际货币基金组织） | 维护国际金融稳定，提供贷款 |
| 世界银行 | 向发展中国家提供贷款和技术援助 |
| APEC（亚太经合组织） | 亚太地区经济合作论坛 |
| RCEP | 区域全面经济伙伴关系协定，2020 年签署，全球最大自贸区 |
| 亚投行（AIIB） | 中国倡议设立，基础设施投资 |

### 三、汇率制度

| 制度 | 说明 |
|------|------|
| 固定汇率制 | 央行维持汇率在固定水平 |
| 浮动汇率制 | 由市场供求决定汇率 |
| 有管理浮动 | 中国现行制度，以市场供求为基础，参考一篮子货币 |

### 四、人民币国际化

- 2016 年人民币正式加入 **IMF 特别提款权（SDR）**货币篮子
- 跨境人民币结算规模持续扩大
- 数字人民币试点推广

## 第六篇：财政与税收

### 一、中国财政体制

| 层级 | 内容 |
|------|------|
| 中央财政 | 国家税务局征收，中央事权支出 |
| 地方财政 | 地方税务局征收，地方事权支出 |
| 转移支付 | 中央对地方的财政补助，促进区域均衡 |

### 二、现行税种（高频🔥）

| 税种 | 性质 | 要点 |
|------|------|------|
| 增值税 | 间接税 | 中国**第一大税种**，多环节征收，可抵扣进项税 |
| 企业所得税 | 直接税 | 税率一般为 **25%**；高新技术企业 **15%** |
| 个人所得税 | 直接税 | 综合所得适用 3%-45% 超额累进税率；起征点 **5000 元/月** |
| 消费税 | 间接税 | 对特定消费品征收（烟、酒、高档化妆品等） |
| 关税 | 间接税 | 进出口环节征收 |

### 三、专项附加扣除（个人所得税）

- 子女教育：每个子女每月 2000 元
- 继续教育：每月 400 元或每年 3600 元
- 大病医疗：每年限额 80000 元
- 住房贷款利息：每月 1000 元
- 住房租金：每月 800-1500 元
- 赡养老人：每月 3000 元
- 3 岁以下婴幼儿照护：每月 2000 元

## 第七篇：金融常识

### 一、金融机构体系

| 机构 | 职能 |
|------|------|
| 中国人民银行 | **中央银行**，制定和执行货币政策 |
| 金融监管总局 | 统一负责除证券业之外的金融业监管 |
| 证监会 | 监管证券期货市场 |
| 商业银行 | 吸收存款、发放贷款、办理结算 |
| 政策性银行 | 国家开发银行、中国进出口银行、中国农业发展银行 |

### 二、货币政策工具对比

| 工具 | 操作 | 效果 |
|------|------|------|
| 降准 | 降低法定存款准备金率 | 增加银行可贷资金→扩张 |
| 降息 | 降低基准利率 | 降低借贷成本→扩张 |
| 公开市场买入 | 央行买入国债 | 投放流动性→扩张 |
| MLF（中期借贷便利） | 央行向商业银行投放中期资金 | 引导中期利率 |
| LPR（贷款市场报价利率） | 银行间报价形成贷款利率基准 | 利率市场化改革成果 |

### 三、股票与基金基础

| 概念 | 要点 |
|------|------|
| 股票 | 股权凭证，收益不固定，风险较高 |
| 债券 | 债权凭证，收益固定，风险较低 |
| 基金 | 集合投资、专业管理、组合投资、利益共享、风险共担 |
| 上市公司 | 经证监会批准在证券交易所上市交易的公司 |

### 四、保险常识

| 类型 | 说明 |
|------|------|
| 社会保险 | 养老、医疗、失业、工伤、生育（五险） |
| 商业保险 | 自愿购买，合同约定 |
| 住房公积金 | 不属于保险，但常与社保并称"五险一金" |

## 第八篇：经济热点专题

### 一、碳达峰碳中和

- **"3060"目标**：2030 年前碳达峰，2060 年前碳中和
- 全国碳排放权交易市场启动
- 绿色金融体系

### 二、反垄断与平台经济

- 反垄断法修订实施
- 平台经济反垄断指南
- 防止资本无序扩张

### 三、乡村振兴

- 产业振兴、人才振兴、文化振兴、生态振兴、组织振兴
- 农村三产融合发展
- 农村土地"三权分置"：所有权、承包权、经营权

### 四、中国现代化产业体系

- **"中国制造 2025"** — 制造强国战略
- 战略性新兴产业：新一代信息技术、高端装备制造、新材料、生物医药等
- **专精特新**：专业化、精细化、特色化、新颖化中小企业
