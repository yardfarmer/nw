## 第一篇：中国历史

### 一、重要朝代与制度

| 朝代 | 关键制度/事件 |
|------|---------------|
| 夏 | 中国第一个王朝；世袭制取代禅让制 |
| 商 | 甲骨文（最早的成熟文字）；青铜器鼎盛 |
| 西周 | 分封制、宗法制、礼乐制度 |
| 秦 | 统一六国；郡县制；统一文字、货币、度量衡 |
| 汉 | 察举制（选官）；丝绸之路；罢黜百家独尊儒术 |
| 隋 | 科举制；三省六部制；大运河 |
| 唐 | 科举制完善；贞观之治/开元盛世 |
| 宋 | 活字印刷术（毕昇）；重文轻武 |
| 元 | 行省制度（省制开端）；宣政院（管理西藏） |
| 明 | 内阁制；八股取士；郑和下西洋 |
| 清 | 军机处（君主专制顶峰）；改土归流 |

### 二、近代史重点（必背⭐）

| 事件 | 时间 | 意义 |
|------|------|------|
| 鸦片战争 | 1840 年 | 中国近代史开端 |
| 太平天国 | 1851-1864 | 农民运动最高峰 |
| 洋务运动 | 1860s-1890s | "中体西用"，自强求富 |
| 戊戌变法 | 1898 年 | 改良失败，思想启蒙 |
| 辛亥革命 | 1911 年 | 推翻帝制，建立民国 |
| 新文化运动 | 1915 年起 | 民主与科学 |
| 五四运动 | 1919 年 | 新民主主义革命开端 |
| 中共成立 | 1921 年 | 开天辟地 |
| 遵义会议 | 1935 年 | 生死攸关的转折点 |
| 抗战胜利 | 1945 年 | 近代以来第一次完全胜利 |
| 新中国成立 | 1949 年 | 人民当家作主 |

### 三、重要不平等条约

| 条约 | 内容 |
|------|------|
| 《南京条约》 | 割让香港岛、五口通商、赔款 |
| 《马关条约》 | 割台湾及澎湖列岛、允许设厂 |
| 《辛丑条约》 | 赔款最多、完全沦为半殖民地半封建社会 |

## 第五篇：中国哲学思想

### 一、诸子百家（必背⭐）

| 学派 | 代表人物 | 核心主张 |
|------|----------|----------|
| 儒家 | 孔子、孟子、荀子 | 仁、礼、德治、仁政；孟子性善论、荀子性恶论 |
| 道家 | 老子、庄子 | 道法自然、无为而治、齐物论 |
| 法家 | 韩非子、商鞅、李斯 | 以法治国、严刑峻法、君主集权 |
| 墨家 | 墨子 | 兼爱、非攻、尚贤、节用 |
| 兵家 | 孙武、孙膑 | 《孙子兵法》——"知己知彼，百战不殆" |
| 纵横家 | 苏秦、张仪 | 合纵连横 |
| 名家 | 公孙龙 | "白马非马"——名实之辩 |
| 杂家 | 吕不韦 | 兼收并蓄，博采众长 |

### 二、儒家发展脉络

| 阶段 | 人物 | 贡献 |
|------|------|------|
| 创立 | 孔子 | 仁、礼、有教无类 |
| 发展 | 孟子 | 仁政、民贵君轻、性善论 |
| 发展 | 荀子 | 礼法并用、性恶论 |
| 汉代 | 董仲舒 | "罢黜百家，独尊儒术"，天人感应 |
| 宋明 | 程朱理学 | 朱熹"存天理，灭人欲"，格物致知 |
| 宋明 | 陆王心学 | 王阳明"知行合一""致良知" |

### 三、古代重要思想命题

| 命题 | 出处/人物 | 含义 |
|------|-----------|------|
| 天人合一 | 儒道共有 | 人与自然和谐统一 |
| 阴阳五行 | 《周易》 | 阴阳对立统一；金木水火土相生相克 |
| 中庸之道 | 孔子 | 不偏不倚，过犹不及 |
