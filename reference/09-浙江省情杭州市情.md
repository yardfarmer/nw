| 考点 | 内容 |
|------|------|
| **"八八战略"** | 习近平 2003 年在浙江提出，浙江发展总纲领 |

| **共同富裕示范区** | 2021 年中央赋予浙江高质量发展建设共同富裕示范区使命 |
| **"两山"理念** | "绿水青山就是金山银山"，发源于浙江**安吉余村** |
| **"枫桥经验"** | 发源于浙江**诸暨枫桥**，"小事不出村、大事不出镇、矛盾不上交" |
| **数字化改革** | 浙江率先推进数字化改革，"数字浙江"建设走在全国前列 |

## 第二篇：杭州市情

### 一、基本概况

| 项目 | 内容 |
|------|------|
| 定位 | 浙江省会、副省级市、长三角中心城市 |

| 建德市 | 唯一县级市 |

### 三、核心考点（必背⭐🔥）

| 考点 | 内容 |
|------|------|
| 数字经济第一城 | 数字经济核心产业增加值占 GDP 约 28%，全国领先 |
| 电商之都 | 阿里巴巴总部所在地 |
| 2023 年亚运会 | 第 19 届亚运会，继北京、广州后第三个举办亚运会的中国城市 |
| 2016 年 G20 峰会 | G20 领导人第十一次峰会 |

2. 大运河（2014 年）
3. 良渚古城遗址（2019 年）
