## 一、考试定位（先读这一篇）

**杭州区县事业编考试（临安、萧山、余杭、富阳、淳安、临平、桐庐、建德等）不分类**，所有岗位统一试卷，区别于浙江省统考 ABCDE 五类。科目为《综合基础知识》或《综合基础知识与写作》/《综合基础知识与综合应用能力》。

- 单科考试，时长 2–2.5 小时
- 偏重记忆与基础知识，部分试卷含主观题（材料写作 / 策论文，30–40 分）
- 题型固定：单选 30 题×1 分 + 多选 10 题×2 分 + 判断 10 题×1 分 + 主观题 1–2 题（合计 30–40 分）

详见 [`00-考试说明/考试大纲与备考策略.md`](./00-考试说明/考试大纲与备考策略.md)。

## 三、标注约定

| 标注 | 含义 |
|------|------|
| ⭐ | S 级必背考点（出现频率 ≥ 30%） |
| 🔥 | A 级常考考点（出现频率 15–30%） |
| ※ | B 级了解考点（出现频率 < 15%，但近年新增） |
| 💡 | 速记口诀 / 记忆技巧 |
| ⚠️ | 易错点 / 陷阱 |
| 📜 | 真题原题（标注年份与考区） |

## 四、备考路径建议（3 阶段）

### 阶段 1：基础打底（4–6 周）

1. 每日通读 1 个模块的 [`知识点.md`](./01-时政热点/知识点.md)，配合 [`速记口诀.md`](./01-时政热点/速记口诀.md) 背诵
2. 用 [`速记卡/`](./速记卡/) 自测，每天 30 张
3. 完成 [`题库/`](./题库/) 中该模块的"基础练习"

### 阶段 2：强化突破（3–4 周）

1. 攻克 [`易混对比.md`](./03-法律常识/易混对比.md)，消灭易错点
2. 刷 [`高频真题.md`](./01-时政热点/高频真题.md)，复盘错题
3. 写作模块每日练 1 篇，套用 [`答题模板/`](./答题模板/)

### 阶段 3：冲刺模考（2 周）

1. 整套真题限时模考（用 [`题库/真题套卷/`](./题库/真题套卷/)）
2. 重点回看 ⭐ 标注的 S 级考点
3. 时政模块刷新 [`2026更新.md`](./01-时政热点/2026更新.md)
