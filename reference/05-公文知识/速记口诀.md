## 一、15 种法定公文

```
合计：15 种
```

## 二、按用途分类口诀

### 领导指导性 → 决决命批

```
决议 — 会议通过的重大决策
决定 — 重要事项部署、奖惩
命令（令）— 公布法规、强制措施
批复 — 答复请示
```

### 呈报性 → 报请

```
报告 — 汇报工作、反映情况
请示 — 请求批准、指示
```

### 公布性 → 公公告通

```
公报 — 重要决定或重大事项
公告 — 国内外宣布重要事项
通告 — 辖区公布应遵守事项
```

### 商洽性 → 函

```
函 — 不相隶属机关之间商洽
```

### 会议性 → 纪要

```
纪要 — 记载传达会议情况
```

### 通用性 → 意通通议

```
意见 — 灵活（上行/下行/平行均可）
通知 — 用途最广
通报 — 表彰/批评
议案 — 政府向人大提请审议
```

## 三、易混文种对比口诀

### 公告 vs 通告 → 公告国外、通告辖区

```
公告 — 国内外、高级别、重大事项
通告 — 辖区内、各级行政、应遵守事项
公报 — 重要决定、重大事项
```

### 通知 vs 通报 → 通知事前、通报事后

```
通知 — 事前告知，要执行
通报 — 事后表彰批评，起教育
```

### 报告 vs 请示 → 报告不批、请示必批

```
报告 — 事后、汇报、不必批复、可多事
请示 — 事前、请求批准、必须批复、一文一事
```

### 请示 vs 函 → 请示上行、函平行

```
请示 — 上行文（下级对上级）
函 — 平行文（不相隶属）
```

### 决议 vs 决定 → 决议会议、决定直接

```
决议 — 会议讨论通过
决定 — 领导机关直接作出
```

### 意见 → 三向均可

```
意见 — 上行 / 下行 / 平行均可（最灵活）
```

## 四、公文格式三部分

### 版头主体版记 → 三段式

```
版头 — 份号、密级、紧急程度、发文机关标志、发文字号、签发人
主体 — 标题、主送机关、正文、附件说明、署名、成文日期、印章
版记 — 抄送机关、印发机关、印发日期
```

### 版头口诀 → 份密紧、发文签

```
份号（涉密必标）
密级（绝密★30、机密★20、秘密★10）
紧急程度（特急、加急）
发文机关标志（红色大字）
发文字号（机关代字+〔年份〕+序号）
签发人（上行文必标）
```

### 主体口诀 → 标主正附、署日印

```
标题（机关+事由+文种）
主送机关（顶格+冒号）
正文（核心内容）
附件说明
署名（发文机关）
成文日期（阿拉伯数字，右空四字）
印章（除纪要外都盖）
```

### 版记口诀 → 抄印发

```
抄送机关
印发机关
印发日期
```

## 五、发文字号

### 发文字号 → 代字+〔年份〕+序号

```
机关代字 + 六角括号〔〕年份 + 序号
例：国发〔2025〕1 号
```

⚠️ 必须是六角括号〔〕，不是 []()【】

## 六、成文日期

### 成文日期 → 阿拉伯数字、不编虚位

```
正确：2025 年 3 月 5 日
错误：2025 年 03 月 05 日
错误：二〇二五年三月五日
```

### 成文日期位置 → 右空四字

```
成文日期右空四字编排
印章居中下压成文日期
```

## 七、行文规则

### 上行文规则 → 一主多抄不下级

```
原则主送一个上级机关
可抄送相关上级和同级
不抄送下级机关
必须标注签发人
```

### 下行文规则 → 主送抄送及上级

```
主送受理机关
可抄送相关机关
重要行文抄送直接上级
```

### 联合行文 → 同级才能联合

```
同级党政机关可联合
同级政府各部门可联合
不相隶属不能联合
```

### 请示规则 → 一文一事不夹带

```
一文一事
不得在报告中夹带请示
不得越级请示
不得抄送下级
请示事项涉他部门应协商一致
```

### 报告规则 → 不夹带请示

```
不得夹带请示事项
不得越级报告
根据需要抄送上级和同级
```

## 八、公文处理流程

### 发文办理 → 拟审签印发

```
拟稿 → 审核 → 签发 → 印制 → 发出
```

### 收文办理 → 签初承传催答

```
签收 → 登记 → 初审 → 承办 → 传阅 → 催办 → 答复
```

💡 **简化**：**签登初承传催答**

### 整理归档 → 及时归档

```
办理完毕 → 及时整理归档
个人不得保存应归档公文
```

## 九、公文用纸与字体

### 用纸 → A4

```
210mm × 297mm
上 37、下 35、左 28、右 26
每页 22 行，每行 28 字
```

### 字体字号

```
发文机关标志 — 红色小标宋体
标题 — 2 号小标宋体
正文 — 3 号仿宋体
一级标题 — 3 号黑体
二级标题 — 3 号楷体
页码 — 4 号半角宋体阿拉伯数字
```

### 正文层次序数 → 一、（一）1.（1）

```
第一层 — 一、（黑体）
第二层 — （一）（楷体）
第三层 — 1.（仿宋体）
第四层 — （1）
```

## 十一、印章规则

### 印章 → 除纪要外都盖

```
公文除纪要和有特定标志的公文外，均加盖印章
印章端正、居中下压成文日期
```

## 十二、标题结构

### 标题 → 机关+事由+文种

```
发文机关 + 关于 + 事由 + 的 + 文种
例：国务院关于印发 2026 年政府工作报告的通知
```

### 标题标点 → 一般不加、法规加书名号

```
公文标题一般不用标点
法规、规章名称加书名号《》
事由中并列成分用顿号
```

## 十三、易错点速查

| 易错点 | 正确 |
|--------|------|
| 发文字号年份括号 | 六角括号〔〕 |
| 成文日期 | 阿拉伯数字不编虚位 |
| 请示能否多事 | 一文一事 |

| 上行文是否标注签发人 | 必须 |

| 公告面向范围 | 国内外 |
| 通告面向范围 | 辖区内 |
| 通知事态 | 事前 |
| 通报事态 | 事后 |

## 十五、事业单位公文特点

### 主要文种 → 通报请函纪意

```
通知、报告、请示、函、纪要、通报、意见
一般不用命令、决定、决议等高级别文种
```

### 常见错误 → 一文多事、越级、夹带

```
请示一文多事 → 错（应一文一事）
越级请示 → 错（一般不得越级）
报告夹带请示 → 错（应单独行文）
用请示反映情况 → 错（应用报告）
函向不相隶属机关请示 → 对（用请批函）
标题缺文种 → 错（必须含文种）
```
