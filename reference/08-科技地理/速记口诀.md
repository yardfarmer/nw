## 二、基础科学

### 物理三大力学 → 牛二定、万引、作用反

```
牛顿第一定律 — 惯性定律（物体保持静止或匀速）
牛顿第二定律 — F = ma
牛顿第三定律 — 作用力与反作用力大小相等方向相反
```

### 物理量单位 → 力 N、能 J、功率 W、压强 Pa

```
力 — 牛顿 N
能量/功 — 焦耳 J
功率 — 瓦特 W
压强 — 帕斯卡 Pa
电量 — 库仑 C
电压 — 伏特 V
电阻 — 欧姆 Ω
频率 — 赫兹 Hz
```

### 物态变化 → 固液气六变

```
熔化 — 固→液（吸热）
凝固 — 液→固（放热）
汽化 — 液→气（吸热，蒸发+沸腾）
液化 — 气→液（放热）
升华 — 固→气（吸热，干冰、樟脑）
凝华 — 气→固（放热，霜、雪）
```

💡 升华凝华不经过液态。

### 化学元素 → HONC、最丰元素

```
宇宙最丰 — 氢 H（75%）、氦 He（24%）
地壳最丰 — 氧 O（48.6%）、硅 Si、铝 Al（金属第一）
人体最丰 — 氧 O、碳 C、氢 H、氮 N
空气最丰 — 氮 N₂（78%）、氧 O₂（21%）
```

🔥 地壳含量前五：氧硅铝铁钙，金属前二：铝、铁

### pH 值 → 7 中、小酸大碱

```
pH < 7 — 酸性
pH = 7 — 中性
pH > 7 — 碱性
pH 0-14 — 常温
```

💡 pH 每差 1，H⁺ 浓度差 10 倍。

### 生物学基础 → 光合呼吸、DNA 双螺旋

```
光合作用 — 6CO₂ + 6H₂O → C₆H₁₂O₆ + 6O₂（叶绿体，吸光）
呼吸作用 — C₆H₁₂O₆ + 6O₂ → 6CO₂ + 6H₂O + 能量（线粒体）
DNA — 脱氧核糖核酸，双螺旋（沃森克里克 1953）
RNA — 核糖核酸
ATP — 三磷酸腺苷，能量通货
```

### 新能源 → 风光水核生

```
太阳能 — 光伏、光热
风能 — 陆上、海上
水能 — 三峡 22500MW 世界最大
核能 — 华龙一号三代核电
生物质能 — 沼气、生物乙醇
地热能、潮汐能、氢能
```

💡 氢能 — 二次能源、清洁、单位质量能量高
