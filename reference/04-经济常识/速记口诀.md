# 经济常识 · 速记口诀

---

## 一、微观经济学

### 需求定理 → 价升需减、价跌需增

```
需求量与价格反向变动
例外：吉芬、炫耀、投机
```

### 弹性理论 → 缺弹提价、富弹降价

```
Ed < 1 缺乏弹性 → 提价增收（谷贱伤农）
Ed > 1 富有弹性 → 降价增收（薄利多销）
```

### 影响需求弹性的因素 → 替用重时

```
替代品多少
用途多少
重要性（必需品弹性小）
占支出比重
时间长短
```

### 市场结构四类型 → 完竞众同、寡头少数、垄断独一

```
完全竞争 — 众多同质无壁垒
垄断竞争 — 较多差异低壁垒
寡头垄断 — 少数高壁垒
完全垄断 — 一个极高壁垒
```

### 市场失灵四原因 → 公外信垄

```
公共物品 — 非排他非竞争
外部性 — 正（教育）/负（污染）
信息不对称 — 逆向选择事前、道德风险事后
垄断 — 抑制竞争
```

### 科斯定理 → 产权明晰、交易成本为零

```
产权明晰 + 交易成本为零
市场可自发实现资源最优配置
```

## 二、宏观经济学

### 宏观经济四大目标 → 经就物国

```
经济增长
充分就业
物价稳定
国际收支平衡
```

### GDP 三强调 → 境内、最终、市场价值

```
领土范围内（境内）
最终产品（避免重复计算）
市场价值（自给自足不计）
```

### GDP vs GNP → 地 vs 人

```
GDP 看地（境内）
GNP 看人（国民）
GDP = GNP − 国外净要素收入
```

### 通胀类型 → 需成结输

```
需求拉上 — 钱多货少
成本推动 — 工资原料涨
结构性 — 部门不平衡
输入型 — 国外传导
```

### 通胀通缩对策 → 通胀紧、通缩松

```
通胀：紧财政（增税减支）+ 紧货币（升准升息卖债）
通缩：松财政（减税增支）+ 松货币（降准降息买债）
```

### 货币政策 → 买债降准降利率=扩张；卖债升准升利率=紧缩

```
扩张性：买国债、降准、降息
紧缩性：卖国债、升准、升息
```

### 财政政策 → 增支减税=扩张；减支增税=紧缩

```
扩张性：减税、增支、发国债
紧缩性：增税、减支、减国债
```

### 货币供应量 → M0 现金、M1 加活、M2 全加定

```
M0 — 流通中现金
M1 — M0 + 活期存款（现实购买力）
M2 — M1 + 定期+储蓄+其他（现实+潜在）
```

### 恩格尔系数分级 → 贫温康裕

```
> 59%：贫困
50-59%：温饱
40-50%：小康
30-40%：相对富裕
< 30%：富裕
```

### 基尼系数 → 0.4 警戒线

```
< 0.2 绝对平均
0.2-0.3 比较平均
0.3-0.4 相对合理
0.4 警戒线
> 0.5 差距悬殊
```

### PMI → 50 荣枯线

```
> 50：经济扩张
< 50：经济收缩
= 50：荣枯分界
```

### 赤字率与债务率 → 3 / 60

```
赤字率警戒线：3%（占 GDP）
债务率警戒线：60%（占 GDP）
```

### 失业四类型 → 摩结周季

```
摩擦性 — 换工作期间
结构性 — 技能不匹配
周期性 — 经济衰退（不正常）
季节性 — 季节变化
```

### 自然失业率 → 摩擦+结构

```
自然失业率 = 摩擦性 + 结构性失业
不含周期性失业
```

## 四、马克思主义政治经济学

### 商品二因素 → 使价、自社

```
使用价值 — 自然属性（物的有用性）
价值 — 社会属性（无差别的人类劳动）
```

### 劳动二重性 → 具抽

```
具体劳动 — 创造使用价值（人与自然）
抽象劳动 — 创造价值（人与社会）
```

### 价值量决定 → 社会必要劳动时间

```
不是个别劳动时间
不是个别价值
商品按社会价值交换
```

### 劳动生产率与价值量 → 单位反、总量不变

```
单位商品价值量 — 与劳动生产率成反比
同一劳动同一时间创造的价值总量 — 不变
```

### 货币五大职能 → 价流贮支世

```
价值尺度 — 标价（基本）
流通手段 — 现场买卖（基本）
贮藏手段 — 金银储藏
支付手段 — 赊销还债缴税发工资
世界货币 — 国际结算
```

### 货币职能识别口诀 → 标价尺、买卖流、欠债支、藏金贮、国际世

```
标价 → 价值尺度
现场购买 → 流通手段
赊购赊销、还债、缴税、发工资 → 支付手段
金银储藏 → 贮藏手段
国际结算 → 世界货币
```

### 价值规律 → 价围价波动

```
基本内容：价值量由社会必要劳动时间决定，等价交换
表现形式：价格围绕价值上下波动
作用：调节分配、刺激技术、优胜劣汰
```

### 货币流通公式 → 价总 / 流速

```
流通中所需货币量 = 商品价格总额 / 货币流通速度
```

### 资本二分（按价值增值作用） → 不变 c + 可变 v

```
不变资本 c — 购买生产资料，不增值
可变资本 v — 购买劳动力，是剩余价值源泉
```

### 资本二分（按周转方式） → 固定 + 流动

```
固定资本 — 多次转移（机器厂房）
流动资本 — 一次转移（原料、劳动力）
```

### 剩余价值率 → m/v

```
m' = m / v
反映剥削程度
```

### 三种剩余价值生产方法 → 绝延、相提、超率

```
绝对剩余价值 — 延长工作日、提高强度
相对剩余价值 — 全社会生产率提高
超额剩余价值 — 个别资本家率先提高
相对是超额的整体结果
```

### 资本循环 → 货币→生产→商品→货币'

```
货币资本 → 生产资本 → 商品资本 → 货币资本'
```

### 再生产两大部类 → 一生资、二消资

```
第一部类 — 生产资料
第二部类 — 消费资料
简单再生产：I(v+m) = IIc
```

### 资本主义经济危机本质 → 相对过剩

```
生产相对过剩
不是绝对过剩
根本原因：生产社会化 vs 私有制矛盾
```
