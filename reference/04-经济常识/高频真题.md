### 1. 📜 2024 临平·第 10 题

基尼系数是衡量收入差距的重要指标，下列关于基尼系数的说法正确的是（  ）

A. 基尼系数取值范围在 0–1 之间，数值越大表示收入越平均
B. 基尼系数 0.4 是国际公认的警戒线
C. 基尼系数只用于衡量地区间收入差距
D. 基尼系数为零时表示收入绝对不平均

**答案：B**（基尼系数 0.4 是国际公认的警戒线；数值越大越不平等；为 0 表示绝对平均）

## 四、解析与拓展

### 第 1 题解析 — 基尼系数

基尼系数是衡量收入分配差距的指标：
- 取值 0–1
- 0：绝对平均
- 1：绝对不平均
- **0.4 为国际警戒线**
- >0.5 差距悬殊

⚠️ 基尼系数衡量的是收入差距，不限于地区间，也可衡量总体、城乡、行业等。

### 第 2 题解析 — 俗语与经济学原理

| 俗语 | 经济学原理 |
|------|------------|
| 谷贱伤农 | 需求**缺乏弹性**，丰收降价总收入减少 |
| 薄利多销 | 需求**富有弹性**，降价总收入增加 |
| 买的没有卖的精 | **信息不对称** |
| 物以稀为贵 | **供求关系** |
| 一分钱一分货 | 价值决定价格 |

### 第 3 题解析 — 价值量与生产率

🔥 核心规律：
- **单位商品价值量**与**劳动生产率**成**反比**
- **单位商品价值量**与**社会必要劳动时间**成**正比**
- **同一劳动同一时间创造的价值总量**与劳动生产率**无关**

为什么总量不变？生产率提高 1 倍 → 单位商品价值量减半 → 但产量增加 1 倍 → 总价值量不变。

### 第 6 题解析 — GDP 关键点

🔥 GDP 三强调：
1. **领土范围**（不是国民）
2. **最终产品**（不是中间产品，避免重复计算）
3. **市场价值**（自给自足不计入）

不计入当年 GDP 的情形：
- 二手房交易（中介费计入）
- 二手车交易（中介费计入）
- 股票买卖（不创造新价值）
- 政府转移支付（无对应生产）
- 家庭主妇家务（非市场活动）
- 志愿服务

### 第 11 题解析 — 货币职能识别

🔥 五大职能识别：
| 场景 | 职能 |
|------|------|
| 商品标价、定价 | 价值尺度 |
| 现场一手交钱一手交货 | 流通手段 |
| 赊购赊销、还债、缴税、发工资 | 支付手段 |
| 金银被储藏 | 贮藏手段 |
| 国际贸易结算 | 世界货币 |

### 第 12 题解析 — 通胀受益方

🔥 通胀时：
- 受益：债务人（债务实际价值下降）、政府（税收累进、内债贬值）
- 受损：债权人、固定收入者、储蓄者

通缩时反之。

### 第 16 题解析 — 恩格尔系数

恩格尔系数 = 食品支出 / 家庭消费总支出

| 区间 | 水平 |
|------|------|
| >59% | 贫困 |
| 50%–59% | 温饱 |
| 40%–50% | 小康 |
| 30%–40% | 相对富裕 |
| <30% | 富裕 |

### 第 22–23 题解析 — 财政 vs 货币政策工具

| 工具 | 归属 |
|------|------|
| 税收、政府支出、国债、转移支付 | **财政** |
| 存款准备金率、再贴现率、公开市场操作、利率 | **货币** |

💡 口诀：**财管政府钱袋，货币管银行钱袋**。

### 第 27 题解析 — 中国税种核心数据

🔥 必背：
- 增值税：第一大税种，基本税率 13%
- 企业所得税：基本税率 25%，高新 15%
- 个人所得税：综合所得 3%–45% 七级累进，起征点 5000 元/月
- 消费税：特定消费品

## 六、答题技巧

### 单选题

1. **看主体**：GDP vs GNP 看地 vs 看人
2. **看方向**：扩张 vs 紧缩，通胀 vs 通缩
3. **看主体**：财政 vs 货币，政府 vs 央行
4. **看例外**：CPI 不含房价、股票；GDP 不含二手房
5. **看绝对表述**：含"所有""绝对""一律"要警惕

### 多选题

1. **政策工具型**：通常 ABCD 全选
2. **理念要素型**：新发展理念 5 个、专项附加扣除 7 项、供给侧 5 任务
3. **指标含义型**：基尼、恩格尔、PMI、CPI 数据要点
4. **少选得 0.5**：完全不确定的不选

### 判断题

1. **GDP/GNP 辨析**：领土 vs 国民
2. **通胀通缩应对**：紧 vs 松
3. **弹性方向**：单位价值量与生产率成反比
4. **货币职能**：标价=尺度，买卖=流通，欠债=支付
5. **税种性质**：增值税是间接税
6. **2026 数据**：GDP 4.5%–5%，CPI 2% 左右
