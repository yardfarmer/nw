## 一、中国古代史

### 朝代顺序 → 夏商周秦汉三国、晋南北朝隋唐、五代辽宋夏金元明清

```
夏 → 商 → 西周 → 东周（春秋战国）→ 秦 → 西汉 → 东汉
→ 三国（魏蜀吴）→ 西晋 → 东晋 → 南北朝
→ 隋 → 唐 → 五代十国
→ 辽宋夏金 → 元 → 明 → 清
```

### 各朝关键制度

```
夏 — 世袭制
商 — 甲骨文、青铜器
西周 — 分封制、宗法制、礼乐制
秦 — 郡县制、统一文字货币度量衡
汉 — 察举制、丝绸之路
隋 — 科举制、三省六部、大运河
唐 — 贞观之治、开元盛世
宋 — 重文轻武、活字印刷
元 — 行省制度、宣政院
明 — 内阁、八股取士、郑和下西洋
清 — 军机处、改土归流
```

### 三大不平等条约 → 南京割香港、马关割台湾、辛丑半封建

```
南京条约 — 割香港岛、五口通商、近代史开端
马关条约 — 割台湾及澎湖、允许设厂、大大加深半殖民地
辛丑条约 — 赔款最多、完全沦为半殖民地半封建
```

### 近代史重大事件时间 → 1840、1894、1898、1911、1915、1919、1921、1935、1945、1949

```
1840 鸦片战争 — 近代史开端

1898 戊戌变法
1911 辛亥革命 — 推翻帝制
1915 新文化运动
1919 五四运动 — 新民主主义革命开端
1921 中共一大 — 党的诞生
1935 遵义会议 — 生死攸关转折
1945 抗战胜利 — 9.3 纪念日
1949 新中国成立 — 10.1
```
