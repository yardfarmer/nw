## 一、管理学基础

### 管理特征 → 目综动艺二

```
目的性
综合性
动态性
艺术性
二重性（自然 + 社会）
```

### 管理二重性 → 自然 + 社会

```
自然属性 — 生产力、社会化大生产（共性）
社会属性 — 生产关系、社会制度（特殊性）
```

### 管理四职能 → 计组领控

```
计划 — 首要职能
组织 — 基础保障
领导 — 关键环节
控制 — 保障实现
```

### 管理核心 → 决策是核心、计划是首要

```
决策 — 管理的核心
计划 — 管理的首要职能
```

### 控制类型 → 前现反

```
前馈控制 — 事前预防
现场控制 — 事中监督
反馈控制 — 事后纠偏
```

### 管理层次与幅度 → 反比关系

```
幅度大 → 层次少 → 扁平化
幅度小 → 层次多 → 高耸型（金字塔）
```

### 影响管理幅度的因素 → 能力手段规范集中

```
管理者能力强
下属能力强
管理手段先进
工作规范程度高
空间分布集中
→ 与幅度成正比
```

```
概念技能 — 全局、战略（高层最重要）
人际技能 — 沟通协调（所有层次都需要）
```

```
中层 — 人际技能最重要

人际技能 — 所有层次都重要
```

```
人际角色 — 挂名首脑、领导者、联络者

决策角色 — 企业家、混乱驾驭者、资源分配者、谈判者
```

## 四、激励理论

### 马斯洛五层需求 → 生安社尊自

```
1. 生理需求 — 食物、水、睡眠
2. 安全需求 — 人身、健康、就业
3. 社交需求 — 归属、友谊、爱情
4. 尊重需求 — 自尊、他人尊重
5. 自我实现 — 实现潜能、理想
```

### 马斯洛要点 → 低未满、高不动、已满足无激励

```
低层次未满足，高层次不是动力
已满足的需求不再有激励作用
按层次逐级满足
```

### 双因素理论 → 保健不激励、激励才满意

```
保健因素 — 薪酬、环境、人际关系（消除不满，不激励）
激励因素 — 成就、认可、责任、成长（产生满意和激励）
```

### 麦克利兰三需要 → 成权亲

```
成就需要 — 追求卓越
权力需要 — 影响他人
亲和需要 — 友好关系
（三种需要是后天习得）
```

### 期望理论 → M = E × V

```
激励力（M）= 期望值（E）× 效价（V）
期望值 — 努力能否带来绩效
效价 — 回报是否有价值
E 或 V 为 0，M 即为 0
```

### 公平理论 → 横纵向比

```
横向比较 — 与他人比（产出/投入 vs 他人的产出/投入）
纵向比较 — 与自己过去比
```

### SMART 原则 → 具可达相时

```
Specific — 具体
Measurable — 可衡量
Achievable — 可达成
Relevant — 相关
Time-bound — 有时限
```

### 强化理论四方式 → 正负惩消

```
正强化 — 给予奖励，增加行为
负强化 — 撤销不良刺激，增加行为
惩罚 — 给予惩罚，减少行为
消退 — 不给奖励，减少行为
```

⚠️ 负强化 ≠ 惩罚

### 激励理论分类 → 内容、过程、行为改造

```
内容型 — 马斯洛、赫茨伯格、麦克利兰
过程型 — 期望、公平、目标设置
行为改造型 — 强化理论
```

## 五、决策方法与心理效应

### 决策类型 → 战战业

```
战略决策 — 全局长期
战术决策 — 局部中期
业务决策 — 日常短期
```

### 决策类型（重复程度）→ 程非程

```
程序化决策 — 重复、有规可循
非程序化决策 — 新出现、无规可循
```

### 决策类型（环境）→ 确风不

```
确定型 — 结果唯一
风险型 — 多结果，概率可知
不确定型 — 多结果，概率不可知
```

### 头脑风暴法四原则 → 自延追组

```
自由畅想
延迟评判
追求数量
组合改进
```

### 德尔菲法 → 匿名、多轮、共识

```
匿名函询
多轮反馈
专家共识
避免群体压力
```

### SWOT 分析 → 优劣机威

```
S（Strengths）— 内部优势
W（Weaknesses）— 内部劣势
O（Opportunities）— 外部机会
T（Threats）— 外部威胁
```

### PDCA 循环 → 计执检处

```
P（Plan）— 计划
D（Do）— 执行
C（Check）— 检查
A（Act）— 处理（改进）
```

### 决策心理效应 → 沉从晕、霍布森、布里丹

```
沉锚效应 — 先入为主
从众效应 — 随大流
晕轮效应 — 以偏概全
霍布森选择 — 只有一个选项
布里丹之驴 — 犹豫不决
```

### 其他心理效应

```
鸟笼效应 — 引发消费
损失厌恶 — 怕损失
确认偏差 — 选择性关注
黑天鹅 — 罕见重大事件
灰犀牛 — 显见被忽视危机
```


```
全能 → 有限
管制 → 服务
人治 → 法治
传统 → 现代
```

### 服务型政府特征 → 民法透、高公

```
民主责任
法治政府
透明廉洁
高效服务
公正公平
```

### 公共产品特征 → 非排非竞

```
非排他性 — 不能排除他人消费
非竞争性 — 增加消费者不增加成本
```

### 公共产品分类 → 纯、准、俱乐部、资源

```
纯公共产品 — 国防、治安、路灯（完全非排他非竞争）
准公共产品 — 教育、医疗、公路（部分）
俱乐部产品 — 有线电视（排他非竞争）
公共资源 — 公共渔场（非排他竞争）
```

### 4E 原则 → 经效效公

```
Economy — 经济
Efficiency — 效率
Effectiveness — 效益
Equity — 公平
```

### 行政监督 → 内一专职、外权司政党社

```
内部监督：
- 一般监督（上下级）
- 专门监督（审计）
- 职能监督（部门职能范围）

外部监督：
- 权力机关监督（人大）
- 司法监督（法检）
- 政党监督（党内、民主党派）
- 社会监督（舆论、群众）
```

## 七、组织结构

### 组织结构类型 → 直职直事矩网

```
直线制 — 命令统一（小型）
职能制 — 专业分工（专业性强）
直线职能制 — 直线+职能（最常见）
事业部制 — 分权独立（大型）
矩阵制 — 双重领导（项目型）
网络型 — 外包协作（现代）
```

### 直线职能制 → 直线指挥+职能参谋

```
直线部门 — 有指挥权
职能部门 — 参谋，提供建议
最常见
```

### 事业部制 → 分权经营、独立核算

```
按产品/地区/客户划分
分权经营
独立核算
适用大型企业
```

### 矩阵制 → 双重领导

```
职能部门 + 项目组
双重领导
资源灵活
适用项目型组织
```

### 组织设计原则 → 任分统权精、有集稳

```
任务目标原则
分工协作原则
统一指挥原则
权责一致原则
精简高效原则
有效管理幅度原则
集权与分权相结合
稳定性与适应性相结合
```

## 八、领导理论

### 管理方格理论 → 9.9 团队型最佳

```
1.1 贫乏型 — 不闻不问
9.1 任务型 — 只关心任务
1.9 俱乐部型 — 只关心人
5.5 中庸型 — 平衡
9.9 团队型 — 最佳（关心生产+关心人）
```

### 菲德勒权变 → 上下关、任务构、职位权

```
三个情境变量：
上下级关系（好/差）
任务结构（明确/不明确）
职位权力（强/弱）

非常有利或非常不利 → 任务导向型
中等有利 → 关系导向型
```

### 情境领导理论 → 下属成熟度对应风格

```
R1 无能力无意愿 → 指示型
R2 无能力有意愿 → 推销型
R3 有能力无意愿 → 参与型
R4 有能力有意愿 → 授权型
```

### 沟通类型 → 正非、上下平

```
正式 / 非正式
下行（上级→下级）
上行（下级→上级）
平行（同级之间）
```

### 沟通障碍 → 发接渠

```
发送者障碍 — 表达不清
接收者障碍 — 理解偏差
渠道障碍 — 信息失真
```
