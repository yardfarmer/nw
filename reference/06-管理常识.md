# 管理常识

> 考试权重：约 5-8% | 侧重管理学基础理论和公共管理

---

## 一、管理学基础

### 管理职能（必背⭐）

| 职能 | 说明 | 地位 |
|------|------|------|
| 决策 | 选择方案 | 管理的**核心** |
| 计划 | 确定目标和方案 | 管理的**首要职能** |
| 组织 | 配置资源、构建结构 | 基础保障 |
| 领导 | 指挥、协调、激励 | 关键环节 |
| 控制 | 监督、纠偏 | 保障实现 |

### 管理层次与管理幅度

- 两者呈**反比关系**
- 管理幅度越大 → 管理层次越少 → 扁平化组织
- 管理幅度越小 → 管理层次越多 → 金字塔型组织

## 二、管理学理论演进

### 古典管理理论

| 理论 | 人物 | 称号 | 核心观点 |
|------|------|------|----------|
| 科学管理理论 | **泰勒** | 科学管理之父 | 标准化操作、计件工资制 |
| 一般管理理论 | **法约尔** | 现代经营管理之父 | 14 条管理原则、管理五职能 |
| 官僚制理论 | **韦伯** | 组织理论之父 | 理性-合法权威、层级制 |

### 行为科学理论

| 理论 | 人物 | 核心发现 |
|------|------|----------|
| 霍桑实验 | **梅奥** | 工人是"社会人"，非正式组织存在，士气影响效率 |

### 现代管理理论

| 理论 | 核心观点 |
|------|----------|
| 系统理论 | 组织是开放系统 |
| 权变理论 | 没有放之四海而皆准的管理方法 |
| 决策理论 | 管理就是决策（西蒙） |

## 三、激励理论（高频🔥）

### 马斯洛需求层次理论（必背⭐）

从低到高：**生理 → 安全 → 社交 → 尊重 → 自我实现**

- 低层次需求未满足时，高层次需求不会成为主要动力
- 已满足的需求不再有激励作用

### 赫茨伯格双因素理论

| 类型 | 内容 | 效果 |
|------|------|------|
| 保健因素 | 薪酬、工作环境、人际关系 | 消除不满，但不能产生满意 |
| 激励因素 | 成就、认可、责任感、成长 | 产生满意和激励 |

### 期望理论（弗鲁姆）

**激励力 = 期望值 × 效价**

- 期望值：努力能否带来绩效
- 效价：绩效带来的回报是否有价值

### 公平理论（亚当斯）

人们通过与他人比较来判断分配是否公平

## 四、决策方法

| 方法 | 要点 |
|------|------|
| 头脑风暴法 | 自由畅想、延迟评判、追求数量 |
| 德尔菲法 | 匿名函询、多轮反馈、专家共识 |
| 决策树法 | 概率分析、期望值计算 |

### 决策心理效应

| 效应 | 含义 |
|------|------|
| 沉锚效应 | 先入为主，受初始信息影响 |
| 从众效应 | 随大流 |
| 晕轮效应 | 以偏概全 |
| 霍布森选择 | 只有一个选项的"伪选择" |
| 布里丹之驴 | 两个选项各有利弊，犹豫不决 |

## 五、公共管理

### 政府职能

经济调节、市场监管、社会管理、公共服务

### 公共产品

| 特征 | 说明 |
|------|------|
| 非排他性 | 不能排除他人消费 |
| 非竞争性 | 增加消费者不增加成本 |

> 典型公共产品：国防、治安、路灯

### 行政监督体系

| 类型 | 内容 |
|------|------|
| 内部监督 | 上级对下级、审计监督 |
| 外部监督 | 人大监督、司法监督、社会监督（舆论、群众） |

## 六、组织结构类型

| 类型 | 特点 | 适用 |
|------|------|------|
| 直线制 | 命令统一、决策迅速 | 小型组织 |
| 职能制 | 专业化分工 | 专业性强 |
| 直线职能制 | 直线指挥+职能参谋 | 最常见 |
| 事业部制 | 分权经营、独立核算 | 大型企业 |
| 矩阵制 | 双重领导、灵活 | 项目型组织 |
