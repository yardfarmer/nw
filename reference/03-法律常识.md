# 法律常识

> 考试权重：约 20-25% | 宪法 + 民法 + 刑法 + 行政法 + 其他

---

## 第一篇：宪法

### 一、国家基本制度

| 制度 | 内容 |
|------|------|
| 国体（国家性质） | 人民民主专政 |
| 政体（政权组织形式） | 人民代表大会制度（根本政治制度） |
| 根本制度 | 社会主义制度 |
| 基本经济制度 | 公有制为主体、多种所有制经济共同发展 |

### 二、公民基本权利与义务（高频考点🔥）

| 权利/义务 | 说明 |
|-----------|------|
| 平等权 | 公民在法律面前一律平等 |
| 选举权和被选举权 | 年满 **18 周岁**的中国公民，未被剥夺政治权利 |
| 言论、出版等自由 | 言论、出版、集会、结社、游行、示威 |
| 人身自由 | 不受非法逮捕、拘禁、搜查 |
| **劳动权** | 既是权利又是义务 |
| **受教育权** | 既是权利又是义务 |

### 三、国家机构

| 机构 | 性质/要点 |
|------|-----------|
| 全国人大 | 最高国家权力机关；常设机关是全国人大常委会 |
| 国家主席 | 任期每届 5 年 |
| 国务院 | 最高行政机关；实行**总理负责制** |
| 监察委员会 | 国家监察机关；最高监察机关是国家监察委员会 |
| 人民法院 | 国家审判机关；最高审判机关是最高人民法院 |
| 人民检察院 | 国家法律监督机关；最高人民检察院 |

### 四、2018 年修宪要点

- 确立习近平新时代中国特色社会主义思想在国家政治和社会生活中的指导地位
- 健全社会主义法治
- 设立监察委员会
- 国家主席任期修改（删除"连续任职不得超过两届"）

## 第二篇：民法（民法典）

### 一、民事主体

| 类型 | 要点 |
|------|------|
| 自然人 | 具有民事权利能力（始于出生，终于死亡）；民事行为能力分三档 |
| 法人 | 具有民事权利能力和民事行为能力，依法独立享有民事权利和承担民事义务 |
| 非法人组织 | 个人独资企业、合伙企业等 |

### 二、民事行为能力（必背⭐）

| 类型 | 年龄/条件 | 效力 |
|------|-----------|------|
| 完全民事行为能力人 | ≥18 周岁；或 16-18 周岁以自己劳动收入为主要生活来源 | 独立实施民事法律行为 |
| 限制民事行为能力人 | 8-18 周岁；或不能完全辨认自己行为的成年人 | 纯获利行为有效；其他需法定代理人追认 |
| 无民事行为能力人 | <8 周岁；或完全不能辨认自己行为的成年人 | 由法定代理人代理 |

### 三、民事法律行为效力

| 类型 | 情形 | 效力 |
|------|------|------|
| 无效 | 欺诈胁迫损害国家利益、恶意串通、违反强制性规定、违背公序良俗 | 自始无效 |
| 可撤销 | 重大误解、欺诈、胁迫、显失公平 | 需当事人请求撤销 |
| 效力待定 | 限制民事行为能力人实施、无权代理 | 需追认 |

### 四、诉讼时效

- **普通诉讼时效**：3 年
- 最长保护期：20 年
- 不适用诉讼时效：支付抚养费、赡养费、扶养费等

### 五、物权

| 概念 | 要点 |
|------|------|
| 所有权 | 占有、使用、收益、处分四项权能 |
| 用益物权 | 土地承包经营权、建设用地使用权、宅基地使用权、地役权 |
| 担保物权 | 抵押权、质权、留置权 |
| 善意取得 | 无权处分人将动产/不动产转让给受让人时，受让人善意取得所有权 |

### 六、合同

- **要约**：希望与他人订立合同的意思表示
- **承诺**：受要约人同意要约的意思表示
- 合同成立：承诺生效时合同成立

## 第三篇：刑法

### 一、犯罪构成要件（四要件说）

| 要件 | 内容 |
|------|------|
| 犯罪主体 | 实施犯罪的人（自然人/单位） |
| 犯罪主观方面 | 故意（直接/间接）或过失（疏忽大意/过于自信） |
| 犯罪客体 | 刑法所保护而为犯罪所侵害的社会关系 |
| 犯罪客观方面 | 危害行为、危害结果、因果关系 |

### 二、刑事责任年龄（必背⭐）

| 年龄段 | 刑事责任 |
|--------|----------|
| <12 周岁 | 不负刑事责任 |
| 12-14 周岁 | 犯故意杀人、故意伤害罪，致人死亡或特别残忍手段致人重伤造成严重残疾，经最高检核准追诉的，负刑事责任 |
| 14-16 周岁 | 犯故意杀人、故意伤害致人重伤或死亡、强奸、抢劫、贩卖毒品、放火、爆炸、投放危险物质罪的，负刑事责任 |
| ≥16 周岁 | 完全负刑事责任 |

### 三、正当防卫与紧急避险

| 概念 | 条件 |
|------|------|
| 正当防卫 | 不法侵害正在进行；针对不法侵害人本人；没有明显超过必要限度 |
| 紧急避险 | 危险正在发生；迫不得已；损害较小合法权益保护较大合法权益 |

### 四、刑罚

| 类型 | 内容 |
|------|------|
| 主刑 | 管制、拘役、有期徒刑、无期徒刑、死刑 |
| 附加刑 | 罚金、剥夺政治权利、没收财产、驱逐出境 |

## 第四篇：行政法

### 一、行政行为分类

| 类型 | 说明 |
|------|------|
| 抽象行政行为 | 制定法规、规章等具有普遍约束力的行为 |
| 具体行政行为 | 针对特定对象的具体处理决定 |

### 二、行政处罚（高频考点🔥）

| 处罚种类 | 内容 |
|----------|------|
| 声誉罚 | 警告、通报批评 |
| 财产罚 | 罚款、没收违法所得、没收非法财物 |
| 行为罚 | 暂扣许可证件、降低资质等级、吊销许可证件、限制开展生产经营活动、责令停产停业、责令关闭、限制从业 |
| 人身自由罚 | 行政拘留 |

**行政处罚设定权：**

| 机关 | 可设定的处罚 |
|------|-------------|
| 法律 | 所有种类 |
| 行政法规 | 除限制人身自由外的处罚 |
| 地方性法规 | 除限制人身自由、吊销营业执照外的处罚 |

### 三、行政许可

- 设定：法律、行政法规、地方性法规可设定；省级政府规章可临时设定
- 原则：许可法定、公开公平公正、便民高效

### 四、行政复议与行政诉讼

| 对比 | 行政复议 | 行政诉讼 |
|------|----------|----------|
| 性质 | 行政救济 | 司法救济 |
| 受理机关 | 上级行政机关 | 人民法院 |
| 审查范围 | 合法性+合理性 | 原则上只审查合法性 |
| 时效 | 60 日 | 6 个月 |
| 程序 | 一级复议制 | 两审终审 |

## 第五篇：其他重要法律

### 劳动法要点

- 试用期最长不超过 6 个月（合同 3 年以上或无固定期限）
- 加班工资：平时 1.5 倍，休息日 2 倍，法定节假日 3 倍
- 经济补偿金：每满一年支付一个月工资

### 公务员法要点

- 公务员范围：依法履行公职、纳入行政编制、由国家财政负担工资福利
- 考核：平时考核 + 年度考核（优秀、称职、基本称职、不称职）
- 处分：警告、记过、记大过、降级、撤职、开除
