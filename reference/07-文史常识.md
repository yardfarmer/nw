## 第一篇：中国历史

### 一、重要朝代与制度

| 朝代 | 关键制度/事件 |
|------|---------------|
| 夏 | 中国第一个王朝；世袭制取代禅让制 |
| 商 | 甲骨文（最早的成熟文字）；青铜器鼎盛 |
| 西周 | 分封制、宗法制、礼乐制度 |
| 秦 | 统一六国；郡县制；统一文字、货币、度量衡 |
| 汉 | 察举制（选官）；丝绸之路；罢黜百家独尊儒术 |
| 隋 | 科举制；三省六部制；大运河 |
| 唐 | 科举制完善；贞观之治/开元盛世 |
| 宋 | 活字印刷术（毕昇）；重文轻武 |
| 元 | 行省制度（省制开端）；宣政院（管理西藏） |
| 明 | 内阁制；八股取士；郑和下西洋 |
| 清 | 军机处（君主专制顶峰）；改土归流 |

### 二、近代史重点（必背⭐）

| 事件 | 时间 | 意义 |
|------|------|------|
| 鸦片战争 | 1840 年 | 中国近代史开端 |
| 太平天国 | 1851-1864 | 农民运动最高峰 |
| 洋务运动 | 1860s-1890s | "中体西用"，自强求富 |
| 戊戌变法 | 1898 年 | 改良失败，思想启蒙 |
| 辛亥革命 | 1911 年 | 推翻帝制，建立民国 |
| 新文化运动 | 1915 年起 | 民主与科学 |
| 五四运动 | 1919 年 | 新民主主义革命开端 |
| 中共成立 | 1921 年 | 开天辟地 |
| 遵义会议 | 1935 年 | 生死攸关的转折点 |
| 抗战胜利 | 1945 年 | 近代以来第一次完全胜利 |
| 新中国成立 | 1949 年 | 人民当家作主 |

### 三、重要不平等条约

| 条约 | 内容 |
|------|------|
| 《南京条约》 | 割让香港岛、五口通商、赔款 |
| 《马关条约》 | 割台湾及澎湖列岛、允许设厂 |
| 《辛丑条约》 | 赔款最多、完全沦为半殖民地半封建社会 |
