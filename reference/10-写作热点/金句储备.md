## 一、习近平总书记重要论述 ⭐

### 一、共同富裕

```
1. 共同富裕是社会主义的本质要求，是中国式现代化的重要特征。
2. 让发展成果更多更公平惠及全体人民。
3. 共同富裕不是少数人的富裕，也不是整齐划一的平均主义。
4. 在高质量发展中促进共同富裕。
5. 幸福都是奋斗出来的。
```

### 二、新发展理念

```
1. 创新、协调、绿色、开放、共享的新发展理念。
2. 把握新发展阶段，贯彻新发展理念，构建新发展格局。
3. 高质量发展是全面建设社会主义现代化国家的首要任务。
4. 发展新质生产力是推动高质量发展的内在要求和重要着力点。
```

### 三、生态文明

```
1. 绿水青山就是金山银山。
2. 像保护眼睛一样保护生态环境，像对待生命一样对待生态环境。
3. 生态兴则文明兴，生态衰则文明衰。
4. 良好生态环境是最普惠的民生福祉。
5. 共建地球生命共同体。
```

### 四、以人民为中心

```
1. 人民对美好生活的向往，就是我们的奋斗目标。
2. 江山就是人民，人民就是江山。
3. 以人民为中心的发展思想。
4. 群众利益无小事。
5. 民之所忧，我必念之；民之所盼，我必行之。
```

### 五、改革开放

```
1. 改革开放是决定当代中国命运的关键一招。
2. 改革永远在路上，只有进行时，没有完成时。
3. 中国开放的大门只会越开越大。
4. 改革是推动发展的强大动力。
```

### 六、全面从严治党

```
1. 打铁必须自身硬。
2. 全面从严治党永远在路上。
3. 不忘初心，牢记使命。
4. 时代是出卷人，我们是答卷人，人民是阅卷人。
```

### 七、科技创新

```
1. 科技是第一生产力，人才是第一资源，创新是第一动力。
2. 关键核心技术是要不来、买不来、讨不来的。
3. 把科技自立自强作为国家发展的战略支撑。
4. 创新是引领发展的第一动力。
```

### 八、乡村振兴

```
1. 民族要复兴，乡村必振兴。
2. 农业农村优先发展。
3. 产业兴旺、生态宜居、乡风文明、治理有效、生活富裕。
4. 粮食安全是国之大者。
```

### 九、文化自信

```
1. 文化自信是更基础、更广泛、更深厚的自信。
2. 没有高度的文化自信，没有文化的繁荣兴盛，就没有中华民族伟大复兴。
3. 推动中华优秀传统文化创造性转化、创新性发展。
4. 以文化人，以文育人。
```

### 十、实干精神

```
1. 空谈误国，实干兴邦。
```

## 十、金句使用原则

### 一、用对地方

```
开头 — 引入金句，引出主题
对策段 — 点缀金句，增强说服力
结尾 — 升华金句，呼应主题
```

### 二、用得恰当

```
- 紧扣主题，不生搬硬套
- 准确引用，不张冠李戴
- 适度使用，不堆砌
```

### 三、用得新颖

```
- 新讲话优于旧讲话
- 浙江特色金句优于通用金句
- 古典金句与时代结合
```
