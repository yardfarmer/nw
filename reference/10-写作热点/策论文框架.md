# 策论文框架

---

## 一、五段式标准框架（推荐）⭐

### 框架结构

```
第一段（150-200 字）— 引论
  开头引入 + 提出观点
  
第二段（200-250 字）— 是什么/为什么
  分析问题 + 阐述重要性
  
第三段（200-250 字）— 怎么办一
  分论点 1 + 论据 + 对策
  
第四段（200-250 字）— 怎么办二
  分论点 2 + 论据 + 对策
  
第五段（150-200 字）— 结论
  总结 + 提升 + 呼吁
```

### 示范：以"共同富裕"为主题

**标题**：以共同富裕绘就浙江新画卷

**第一段（引论）**：
> 共同富裕是社会主义的本质要求，是中国式现代化的重要特征。2021 年中央部署浙江高质量发展建设共同富裕示范区，赋予浙江光荣使命。新征程上，我们要深化八八战略，破解发展难题，扎实推进共同富裕，让发展成果更多更公平惠及全体人民。

**第二段（分析）**：
> 当前，浙江共同富裕示范区建设取得阶段性成果，城乡居民收入倍差缩小至 1.83，但城乡、区域、收入三大差距依然存在，扩中提低任务艰巨。究其原因，既有发展不平衡不充分问题，也有体制机制不完善因素。这要求我们必须以更大力度、更实举措推进共同富裕。

**第三段（对策一）**：
> **深化"扩中提低"改革，缩小收入差距**。重点是提高低收入群体收入，扩大中等收入群体规模。一是完善初次分配机制，提高劳动报酬占比；二是健全再分配机制，强化税收、社保、转移支付调节；三是鼓励三次分配，发展慈善事业。通过"扩中提低"，让更多人共享发展成果。

**第四段（对策二）**：
> **加快山区 26 县高质量发展，缩小区域差距**。山区 26 县是浙江共富短板。一是实施"一县一策"，因地制宜发展特色产业；二是深化山海协作，推进产业飞地建设；三是加强基础设施和公共服务补短板。让山区 26 县与全省同步实现现代化。

**第五段（结论）**：
> 潮起钱塘，奋楫者先。共同富裕是一场接力跑，需要我们以"干在实处、走在前列、勇立潮头"的浙江精神，久久为功，扎实推进。让我们以更大力度深化八八战略，推进共同富裕示范区建设，奋力交出中国式现代化浙江高分答卷。

## 二、四段式简洁框架

### 框架结构

```
第一段（200 字）— 引论 + 分析
第二段（300 字）— 对策一
第三段（300 字）— 对策二
第四段（200 字）— 结论
```

适用：字数要求 800-1000 字

## 三、六段式扩展框架

### 框架结构

```
第一段（150 字）— 引论
第二段（200 字）— 分析问题
第三段（200 字）— 对策一
第四段（200 字）— 对策二
第五段（200 字）— 对策三
第六段（150 字）— 结论
```

适用：字数要求 1000-1200 字，对策丰富

## 四、对策段万能模板 ⭐

### 模板一（问题导向）

```
【分论点】。当前，【存在问题】，主要表现在【具体表现】。
究其原因，【原因分析】。因此，要【对策方向】。
一是【对策 1】；二是【对策 2】；三是【对策 3】。
通过【对策】，实现【效果】。
```

### 模板二（目标导向）

```
【分论点】，是【重要性】的必然要求。当前，【现状】。
面向未来，我们要【方向】。具体而言：
第一，【对策 1】；
第二，【对策 2】；
第三，【对策 3】。
让【效果】成为【目标】的生动注脚。
```

### 模板三（意义导向）

```
【分论点】，意义重大。一方面，【意义一】；
另一方面，【意义二】。当前，【现状】。
为此，要【对策方向】。一是【对策 1】；
二是【对策 2】；三是【对策 3】。
以【对策】之笔，绘就【目标】之画卷。
```

必须回答的时代之问。【观点】。
```

【问题】依然存在，与【要求】相比还有差距。
【观点】。
```

```

这要求我们【观点】。
```

为【工作】指明了方向。当前，【背景】，

```

```

## 七、过渡句万能模板

### 段间过渡

```
- 然而，【问题】依然存在。
- 但是，我们也要清醒看到，【问题】。
- 与此同时，【另一方面】。
- 更深层次看，【深层问题】。
- 面对新形势新任务，【对策】。
```

### 对策间过渡

```
- 一是【对策 1】。
- 二是【对策 2】。
- 三是【对策 3】。
- 此外，还要【补充对策】。
- 更重要的是，【核心对策】。
```

### 段内过渡

```
- 一方面，【论点 1】；另一方面，【论点 2】。
- 既要【做法 1】，又要【做法 2】。
- 不仅【层面 1】，更【层面 2】。
- 从【角度 1】看，【分析 1】；从【角度 2】看，【分析 2】。
```

## 十、检查清单

### 提交前检查

- [ ] 标题是否鲜明？

- [ ] 是否有 2-3 个对策段？
- [ ] 对策是否具体可行？
- [ ] 结尾是否呼应开头？
- [ ] 字数是否符合要求？
- [ ] 是否有错别字？
- [ ] 卷面是否整洁？
- [ ] 是否引用了金句或数据？
- [ ] 是否紧扣浙江实际？
