# 写作热点 · 知识点

> 本模块服务事业编考试主观题（材料写作/策论文/案例分析，约 30-40 分）。杭州区县事业编考试常见主观题：材料分析、策论文、公文写作。

> 标注：⭐S 必背 | 🔥A 重点 | ※B 易出题

## 第一篇 主观题题型与评分标准

### 一、题型分类 ⭐

```
1. 策论文 — 围绕给定主题，提出对策建议（最常见）
2. 材料分析题 — 根据材料回答问题，要求归纳、分析、对策
3. 公文写作题 — 起草通知、报告、请示、函、纪要等
4. 案例分析题 — 给出工作场景，分析问题、提出方案
5. 综合写作题 — 给定材料，自拟题目写作
```

### 二、评分标准 🔥

| 维度 | 占比 | 要求 |
|------|------|------|
| 内容 | 40% | 紧扣题意、观点正确、对策可行 |
| 结构 | 20% | 条理清晰、逻辑严密、首尾呼应 |
| 语言 | 20% | 准确流畅、规范得体、无语病 |
| 字数 | 10% | 符合要求（一般 800-1200 字） |
| 卷面 | 10% | 字迹工整、卷面整洁 |

### 三、字数要求 ※

```
材料分析题 — 300-500 字/题
策论文 — 800-1200 字
公文写作 — 300-500 字
案例分析 — 500-800 字
```

## 第二篇 策论文写作框架

### 一、标准结构（五段式）⭐⭐⭐

```
第一段 — 引论（150-200 字）
  引出话题 + 表明观点
  
第二段 — 是什么/为什么（200-250 字）
  分析问题现状 + 重要性
  
第三段 — 怎么办一（200-250 字）
  对策一（分论点+论据+对策）
  
第四段 — 怎么办二（200-250 字）
  对策二（分论点+论据+对策）
  
第五段 — 结论（150-200 字）
  总结提升 + 展望呼吁
```

### 二、标题写法 🔥

#### 1. 对仗式标题

```
深化改革开放，激发发展活力
聚焦共同富裕，破解发展难题
坚持人民至上，办好民生实事
```

#### 2. 主副标题

```
以新质生产力引领高质量发展
——关于推进产业升级的思考
```

#### 3. 主题式标题

```
让"地瓜经济"根深叶茂
共富路上"一个都不能少"
用数字之笔绘就治理新画卷
```

💡 标题要：紧扣主题、观点鲜明、简明有力

### 三、开头写法 ⭐

#### 1. 背景引入式

```
当前，浙江正处于高质量发展建设共同富裕示范区的关键时期。
面对新形势新任务，如何深化"八八战略"，推进省域现代化先行，
成为我们必须回答的时代之问。
```

#### 2. 问题切入式

```
近年来，我省 X 工作取得显著成效，但也要清醒看到，X 仍存在
X 等问题，与高质量发展要求、人民群众期盼相比还有差距。
```

#### 3. 数据引入式

```
2024 年浙江 GDP 突破 9 万亿元，城乡居民收入倍差缩小至 1.83，
但城乡、区域、收入三大差距依然存在。这要求我们 X。
```

### 四、对策段写法 ⭐⭐

#### 对策段万能模板

```
【分论点】。当前，【现状/问题】。【原因分析】。
因此，要【具体对策】。一是【对策 1】；二是【对策 2】；
三是【对策 3】。【预期效果】。
```

#### 对策段示范（数字经济主题）

```
以数字经济赋能高质量发展。当前，我省数字经济核心产业
增加值突破万亿元，但传统产业数字化转型仍存在"不愿转、
不会转、不敢转"等问题。因此，要深化数字经济创新提质
一号工程。一是加强数字基础设施建设，夯实算力、数据、
网络支撑；二是推进产业数字化，建设"产业大脑+未来工厂"；
三是培育数字人才，加强校企合作，打造数字人才高地。
让数字经济成为浙江发展新引擎。
```

### 五、结尾写法 ⭐

#### 1. 总结呼应式

```
潮起钱塘，奋楫者先。新征程上，我们要坚持 X，深化 X，
推进 X，以 X 之力谱写中国式现代化浙江新篇章。
```

#### 2. 展望呼吁式

```
"干在实处、走在前列、勇立潮头"的浙江精神，激励着我们
砥砺前行。让我们以更大力度、更实举措，X，为 X 贡献力量。
```

#### 3. 引用金句式

```
习近平总书记强调："X"。我们要牢记嘱托，X，在 X 之路上
迈出坚实步伐，奋力交出 X 高分答卷。
```

## 第三篇 五大题型应对策略

### 一、材料分析题 ⭐

#### 1. 题型特征

```
给定 1-3 段材料（500-2000 字）
问题 1-3 个：归纳问题/分析原因/提出对策
字数：300-500 字/题
```

#### 2. 答题步骤

```
第一步 — 通读材料，标注关键词（问题、原因、对策）
第二步 — 审题，明确答题方向（归纳/分析/对策）
第三步 — 回读材料，提取相关信息
第四步 — 整合要点，分条作答
第五步 — 检查字数、逻辑
```

#### 3. 答题模板

**归纳问题型**：
```
材料反映了以下 X 方面问题：
一是【问题 1】，表现在【具体表现】；
二是【问题 2】，表现在【具体表现】；
三是【问题 3】，表现在【具体表现】。
```

**分析原因型**：
```
究其原因，主要有：
一是【原因 1】，【分析】；
二是【原因 2】，【分析】；
三是【原因 3】，【分析】。
```

**提出对策型**：
```
针对上述问题，建议：
一是【对策 1】，【具体做法】；
二是【对策 2】，【具体做法】；
三是【对策 3】，【具体做法】。
```

### 二、策论文 🔥

#### 1. 题型特征

```
给定主题或背景材料
要求写一篇策论文
字数：800-1200 字
分值：30-40 分
```

#### 2. 策论文核心要素

```
标题 — 鲜明观点
开头 — 引出问题，表明立场
主体 — 3-4 个对策段，每段一个分论点
结尾 — 总结提升，呼应开头
```

#### 3. 对策拟写"五步法"

```
第一步 — 找问题（材料中的问题是什么）
第二步 — 找原因（为什么会这样）
第三步 — 找方向（应该怎样做）
第四步 — 找措施（具体怎么做）
第五步 — 找效果（达到什么效果）
```

### 三、公文写作题 ⭐

#### 1. 常见公文类型

```
通知 — 最常考
报告 — 较常考
请示 — 较常考
函 — 较常考
纪要 — 偶尔考
```

#### 2. 通知写作模板

```
【标题】关于XXXX的通知

【主送机关】各县（市、区）人民政府，省政府各部门：

【正文】
  为【目的】，根据【依据】，现就【事项】通知如下：
  
  一、【事项 1】。【具体要求】。
  
  二、【事项 2】。【具体要求】。
  
  三、【事项 3】。【具体要求】。
  
  请各地各部门【要求】，确保【效果】。

【落款】XX省人民政府
【日期】2026年X月X日
```

#### 3. 报告写作模板

```
【标题】关于XXXX的报告

【主送机关】XX省人民政府：

【正文】
  根据【依据】，我【单位】开展了【工作】，现将有关情况报告如下：
  
  一、【工作开展情况】
  
  二、【主要成效】
  
  三、【存在问题】
  
  四、【下一步打算】
  
  妥否，请批示。

【落款】XX市（县）人民政府
【日期】2026年X月X日
```

#### 4. 请示写作模板

```
【标题】关于XXXX的请示

【主送机关】XX省人民政府：

【正文】
  为【目的】，根据【依据】，我【单位】拟【事项】。
  
  现就【具体事项】请示如下：
  
  一、【背景情况】
  
  二、【请示事项】
  
  三、【请求】
  
  妥否，请批示。

【落款】XX市（县）人民政府
【日期】2026年X月X日
```

#### 5. 公文写作要点

```
1. 标题 — 三要素（发文机关+事由+文种）
2. 主送机关 — 顶格，全称或规范化简称
3. 正文 — 结构清晰，层次分明
4. 落款 — 发文机关+日期
5. 语言 — 庄重、规范、准确
6. 字数 — 一般 300-500 字
```

### 四、案例分析题 ※

#### 1. 题型特征

```
给定工作场景或案例
要求分析问题、提出方案
字数：500-800 字
```

#### 2. 答题框架

```
第一部分 — 案例反映的问题（150 字）
  归纳案例中存在的主要问题
  
第二部分 — 问题原因分析（200 字）
  分析问题产生的深层原因
  
第三部分 — 解决方案（300-400 字）
  提出可操作的解决方案
  分条列举，每条具体可行
  
第四部分 — 总结（50-100 字）
  简要总结，呼应案例
```

### 五、综合写作题 ※

#### 1. 题型特征

```
给定材料或话题
自拟题目写一篇议论文
字数：800-1200 字
```

#### 2. 写作要点

```
1. 立意 — 紧扣材料，提炼观点
2. 标题 — 鲜明生动
3. 结构 — 凤头、猪肚、豹尾
4. 论证 — 事实论证+理论论证
5. 语言 — 规范、生动、有感染力
```

## 第六篇 写作技巧与避坑

### 一、写作技巧 ⭐

#### 1. 审题"四问"

```
一问 — 写什么文体？（策论文/公文/案例分析）
二问 — 写什么主题？（围绕什么话题）
三问 — 写多少字？（字数要求）
四问 — 给什么材料？（材料怎么用）
```

#### 2. 立意"三看"

```
一看 — 看材料主题
二看 — 看时代背景
三看 — 看浙江实际
```

#### 3. 结构"三要"

```
一要 — 凤头（开头精彩）
二要 — 猪肚（内容充实）
三要 — 豹尾（结尾有力）
```

#### 4. 对策"三性"

```
针对性 — 针对问题提对策
可行性 — 对策要可操作
创新性 — 提出新颖观点
```

### 二、常见错误避坑 🔥

#### 1. 跑题

```
错误 — 偏离材料主题，自说自话
正确 — 紧扣材料，围绕主题
```

#### 2. 对策空泛

```
错误 — "加强管理""完善制度"等空话
正确 — 具体到"加强 XX 管理，建立 XX 制度"
```

#### 3. 结构混乱

```
错误 — 段落无逻辑，前后不衔接
正确 — 总分总结构，分论点清晰
```

#### 4. 语言口语化

```
错误 — "我觉得""大家要""一定要"
正确 — "笔者认为""各级要""必须"
```

#### 5. 字数不足或超字

```
错误 — 字数严重不足或大幅超字
正确 — 严格控制在要求字数±10% 以内
```

#### 6. 卷面潦草

```
错误 — 字迹潦草、涂改严重
正确 — 字迹工整、卷面整洁
```
