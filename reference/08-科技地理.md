## 第二篇：地理常识

### 一、中国地理

#### 行政区划

- **34 个**省级行政区划：23 省 + 5 自治区 + 4 直辖市 + 2 特别行政区

#### 四大高原

| 高原 | 特征 |
|------|------|
| 青藏高原 | 世界屋脊、最高 |
| 内蒙古高原 | 地表平坦 |
| 黄土高原 | 水土流失严重 |
| 云贵高原 | 喀斯特地貌 |

#### 四大盆地

| 盆地 | 特征 |
|------|------|
| 塔里木盆地 | 最大盆地 |
| 准噶尔盆地 | 第二大 |
| 柴达木盆地 | 聚宝盆 |
| 四川盆地 | 紫色盆地/天府之国 |

#### 三大平原

东北平原（最大）、华北平原、长江中下游平原

#### 主要河流

| 河流 | 要点 |
