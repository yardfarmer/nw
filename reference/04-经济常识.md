# 经济常识

> 考试权重：约 10-15% | 侧重宏观经济政策，常与时政结合出题

---

## 第一篇：微观经济学

### 一、供求与价格

| 概念 | 要点 |
|------|------|
| 需求定律 | 价格上升，需求量下降（反向变动） |
| 供给定律 | 价格上升，供给量增加（同向变动） |
| 均衡价格 | 供给量 = 需求量时的价格 |
| 弹性 | 需求价格弹性：需求量变动率 / 价格变动率 |

### 二、市场类型

| 类型 | 厂商数量 | 产品 | 进入壁垒 |
|------|----------|------|----------|
| 完全竞争 | 众多 | 同质 | 无 |
| 垄断竞争 | 较多 | 差异化 | 低 |
| 寡头垄断 | 少数 | 同质或差异化 | 高 |
| 完全垄断 | 一个 | 唯一 | 极高 |

### 三、市场失灵

| 原因 | 说明 |
|------|------|
| 外部性 | 正外部性（教育）和负外部性（污染） |
| 公共物品 | 非排他性 + 非竞争性 |
| 信息不对称 | 逆向选择（事前）、道德风险（事后） |
| 垄断 | 抑制竞争，损害效率 |

## 第二篇：宏观经济学（高频🔥）

### 一、核心指标

| 指标 | 含义 |
|------|------|
| GDP | 国内生产总值——一国境内一定时期生产的最终产品和服务的市场价值 |
| GNP | 国民生产总值——一国国民一定时期生产的最终产品和服务的市场价值 |
| CPI | 居民消费价格指数——衡量通货膨胀的主要指标 |
| 失业率 | 失业人口 / 劳动力人口 |

### 二、通货膨胀与通货紧缩

| 情形 | 表现 | 对策 |
|------|------|------|
| 通货膨胀 | 物价持续上涨、货币贬值 | 紧缩性财政政策 + 紧缩性货币政策 |
| 通货紧缩 | 物价持续下跌、需求不足 | 扩张性财政政策 + 扩张性货币政策 |

### 三、财政政策

| 工具 | 内容 |
|------|------|
| 税收 | 增税→紧缩，减税→扩张 |
| 国债 | 发行国债→扩张（增加政府支出） |
| 政府购买 | 增加购买→扩张 |
| 转移支付 | 增加转移支付→扩张 |

### 四、货币政策

| 工具 | 内容 |
|------|------|
| 法定存款准备金率 | 上调→紧缩（银行可贷资金减少） |
| 再贴现率 | 上调→紧缩（商业银行借款成本增加） |
| 公开市场操作 | 卖出债券→紧缩（回收流动性） |
| 基准利率 | 上调→紧缩（借贷成本增加） |

> 口诀：**"买债降准降利率"= 扩张；"卖债升准升利率"= 紧缩**

### 五、汇率

| 变动 | 影响 |
|------|------|
| 本币升值 | 利于进口，不利于出口；利于出国旅游/留学 |
| 本币贬值 | 利于出口，不利于进口；利于吸引外资 |

## 第三篇：中国经济热点

### 一、新发展理念（必背⭐）

创新、协调、绿色、开放、共享

### 二、新发展格局

以国内大循环为主体、国内国际双循环相互促进

### 三、高质量发展

- 从"有没有"转向"好不好"
- 核心是提高全要素生产率

### 四、共同富裕

- 不是同时富裕，也不是同等富裕
- 不是劫富济贫，而是先富带后富
- 浙江是**高质量发展建设共同富裕示范区**

### 五、新质生产力

- 以科技创新为主导
- 高科技、高效能、高质量
- 绿色发展是底色

### 六、供给侧结构性改革

- 去产能、去库存、去杠杆、降成本、补短板
- 从供给侧发力，提高供给体系质量

## 第四篇：马克思主义政治经济学

| 概念 | 要点 |
|------|------|
| 商品 | 使用价值（自然属性）+ 价值（社会属性） |
| 使用价值 | 满足人的需要的有用性 |
| 价值 | 凝结在商品中的无差别的人类劳动 |
| 价值规律 | 商品价值量由社会必要劳动时间决定；商品交换以价值量为基础 |
| 剩余价值 | 工人创造的超过劳动力价值的那部分价值，被资本家无偿占有 |
